Windows 11 офіційно вимагає 4 ГБ RAM і 8-ядерний процесор, але реально встановлюється і на слабше залізо. Ось як змусити її працювати прийнятно навіть на обмеженому обладнанні.

---

## 1. Вимкни всі візуальні ефекти

Найбільший виграш RAM і CPU на слабких машинах.

`Win + R` → `sysdm.cpl` → **Додатково** → **Параметри** (Швидкодія) → **Налаштувати для найкращої швидкодії**

Або вибери вручну що залишити — рекомендую залишити тільки:
- Згладжування шрифтів екрана
- Показувати мініатюри замість значків

---

## 2. Вимкни прозорість і анімації

`Параметри` → `Персоналізація` → `Кольори` → вимкни **Ефекти прозорості**

`Параметри` → `Спеціальні можливості` → `Візуальні ефекти`:
- Вимкни **Ефекти анімації**
- Вимкни **Прозорість**

---

## 3. Вимкни автозапуск усього зайвого

`Ctrl + Shift + Esc` → **Автозапуск** → вимкни все крім антивірусу і звукових драйверів.

На слабкому ПК навіть OneDrive і Discord у фоні — помітне гальмування.

---

## 4. Збільш файл підкачки

На ПК з 4 ГБ RAM файл підкачки критично важливий.

`Win + R` → `sysdm.cpl` → **Додатково** → **Параметри** → **Додатково** → **Змінити**:
- Зніми **Автоматично вибирати**
- Встанови **Спеціальний розмір**
- Початковий розмір: 4096 МБ
- Максимальний розмір: 8192 МБ

---

## 5. Вимкни Widgets і рекомендації

Widgets (панель новин) завантажує дані у фоні і споживає RAM.

`Параметри` → `Персоналізація` → `Панель завдань` → вимкни **Віджети**

---

## 6. Вимкни індексування пошуку (якщо HDD)

На HDD служба індексування постійно навантажує диск.

`Win + R` → `services.msc` → знайди **Windows Search** → правою кнопкою → **Властивості** → **Тип запуску: Вимкнено** → **Зупинити**

На SSD — залиш увімкненим, там це не проблема.

---

## 7. Вимкни непотрібні фонові служби

```powershell
# Служби що можна вимкнути на слабкому ПК
$services = @(
    "DiagTrack",          # телеметрія
    "SysMain",            # Superfetch (корисний на HDD, але їсть RAM)
    "WSearch",            # пошук (якщо HDD)
    "TabletInputService", # введення з планшета
    "Fax",                # факс
    "XblGameSave",        # Xbox
    "XboxNetApiSvc"       # Xbox мережа
)

foreach ($svc in $services) {
    Set-Service -Name $svc -StartupType Disabled -ErrorAction SilentlyContinue
    Stop-Service -Name $svc -Force -ErrorAction SilentlyContinue
    Write-Output "Вимкнено: $svc"
}
```

---

## 8. Вимкни Game Mode (якщо не граєш)

На слабкому ПК Game Mode іноді навпаки гальмує звичайну роботу.

`Параметри` → `Ігри` → **Режим гри** → вимкни

---

## 9. Очисти диск і тимчасові файли

```powershell
# Швидке очищення тимчасових файлів
Remove-Item "$env:TEMP\*" -Recurse -Force -ErrorAction SilentlyContinue
Remove-Item "C:\Windows\Temp\*" -Recurse -Force -ErrorAction SilentlyContinue

# Запустити очищення диска
cleanmgr /sagerun:1
```

Детальніше: [Як почистити диск C](/yak-pochystyty-dysk-c)

---

## 10. Розглянь перехід на легшу ОС

Якщо після всіх оптимізацій ПК все одно гальмує — Windows 11 просто важка для цього заліза.

Альтернативи:
- **Windows 10** — менш вимоглива, підтримка до жовтня 2025 (продовжено ESU)
- **Linux Mint** — дуже легка, чудова для старих ПК, підходить для перегляду інтернету і документів
- **Lubuntu** — ще легша, для ПК з 2 ГБ RAM

---

## Очікуваний результат

| Дія | Виграш RAM | Виграш CPU |
|-----|-----------|-----------|
| Вимкнути ефекти | ~100 МБ | ~5–10% |
| Вимкнути автозапуск | ~200–500 МБ | ~10–20% |
| Збільшити файл підкачки | стабільність | — |
| Вимкнути служби | ~50–200 МБ | ~3–7% |
| Вимкнути Widgets | ~100 МБ | ~2–5% |

---

## Підсумок

Найбільший ефект: вимкнути візуальні ефекти + автозапуск + збільшити файл підкачки. Разом дають відчутну різницю на ПК з 4 ГБ RAM. Якщо після цього все одно гальмує — проблема в апаратному забезпеченні.
