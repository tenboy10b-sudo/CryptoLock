Більшість людей використовує 5–10 гарячих клавіш. Ще 20 клавіш нижче можуть зекономити годину на день при регулярній роботі за ПК.

---

## Управління вікнами

| Клавіші | Що робить |
|---------|----------|
| `Win + D` | Показати/приховати робочий стіл |
| `Win + E` | Відкрити Провідник |
| `Win + L` | Заблокувати ПК |
| `Win + Tab` | Огляд задач (всі вікна і столи) |
| `Win + ↑` | Розгорнути вікно на весь екран |
| `Win + ↓` | Відновити або згорнути вікно |
| `Win + ←` | Прикріпити вікно ліворуч |
| `Win + →` | Прикріпити вікно праворуч |
| `Win + Shift + ←/→` | Перемістити вікно на інший монітор |
| `Alt + Tab` | Переключитись між вікнами |
| `Alt + F4` | Закрити поточне вікно |
| `Win + число` | Відкрити програму №N з панелі завдань |

---

## Скріншоти

| Клавіші | Що робить |
|---------|----------|
| `Win + PrintScreen` | Скріншот всього екрана → збереження у Pictures\Screenshots |
| `Win + Shift + S` | Вибрати область для скріншоту |
| `Alt + PrintScreen` | Скріншот активного вікна → в буфер |
| `Win + G` | Xbox Game Bar (скріншот, запис відео) |

---

## Робота з текстом

| Клавіші | Що робить |
|---------|----------|
| `Ctrl + Z` | Скасувати |
| `Ctrl + Y` | Повторити |
| `Ctrl + A` | Виділити все |
| `Ctrl + C / X / V` | Копіювати / Вирізати / Вставити |
| `Ctrl + Shift + V` | Вставити без форматування (в браузері) |
| `Win + V` | Буфер обміну з історією (25 останніх) |
| `Ctrl + Home / End` | На початок / кінець документа |
| `Ctrl + ← / →` | Переміститись на одне слово |
| `Ctrl + Shift + ← / →` | Виділити слово |
| `F2` | Перейменувати виділений файл |

| Клавіші | Що робить |
|---------|----------|

| `Win + I` | Параметри Windows |

| `Win + S` | Пошук |

| `Win + K` | Підключення (Bluetooth, дисплеї) |

| `Win + Pause` | Властивості системи |
| `Win + .` | Меню емодзі і символів |

## Віртуальні робочі столи

| Клавіші | Що робить |
|---------|----------|
| `Win + Ctrl + D` | Новий робочий стіл |
| `Win + Ctrl + →` | Наступний стіл |
| `Win + Ctrl + ←` | Попередній стіл |
| `Win + Ctrl + F4` | Закрити поточний стіл |

## Підсумок

П'ять клавіш що варто вивчити першими: `Win + Shift + S` (скріншот), `Win + V` (буфер обміну), `Win + L` (блокування), `Ctrl + Shift + Esc` (диспетчер завдань), `Win + D` (робочий стіл). Решта засвоюються поступово при регулярному використанні.
