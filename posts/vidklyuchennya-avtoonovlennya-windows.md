Автоматичні оновлення Windows іноді заважають — перезавантажують ПК в невідповідний момент, займають трафік або ламають щось що працювало. Розберемо як це контролювати.

## Спосіб 1: Відкласти оновлення (рекомендовано)

Це найбезпечніший варіант — оновлення не вимикаються назавжди, просто відкладаються.

**Windows 11:**
`Win + I` → **Windows Update** → **Додаткові параметри** → **Призупинити оновлення** → вибери на скільки тижнів.

**Windows 10:**
`Win + I` → **Оновлення і безпека** → **Додаткові параметри** → **Призупинити оновлення**.

Максимум можна призупинити на 35 днів, після чого потрібно дозволити встановлення хоч одного оновлення.

---

## Спосіб 2: Через групову політику (повне вимкнення)

**Крок 1.** `Win + R` → `gpedit.msc` → Enter.

**Крок 2.** Перейди:
```
Конфігурація комп'ютера
  → Адміністративні шаблони
    → Компоненти Windows
      → Windows Update
        → Керування оновленнями, що надходять від Windows Update
```

**Крок 3.** Знайди **Настройка автоматичного оновлення** → двічі клікни → **Включено**.

**Крок 4.** У розділі **Параметри** вибери **2 — Сповіщати про завантаження і встановлення**.

Тепер оновлення не встановлюватимуться автоматично — тільки покаже сповіщення.

Або вибери **1 — Вимкнути автоматичне оновлення** — для повного вимкнення.

---

## Спосіб 3: Зупинити службу Windows Update

**Крок 1.** `Win + R` → `services.msc` → Enter.

**Крок 2.** Знайди **Windows Update** у списку → двічі клікни.

**Крок 3.** 
- **Тип запуску:** Вимкнено
- Натисни **Зупинити**
- OK

**Крок 4.** Щоб Windows не запускала службу знову — через CMD:

```cmd
sc config wuauserv start= disabled
sc stop wuauserv
```

---

## Спосіб 4: Через реєстр

```cmd
reg add "HKLM\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate\AU" /v NoAutoUpdate /t REG_DWORD /d 1 /f
reg add "HKLM\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate\AU" /v AUOptions /t REG_DWORD /d 1 /f
```

---

## Як налаштувати оновлення тільки в певний час

Замість повного вимкнення — налаштуй активні години коли Windows не буде перезавантажуватись:

`Win + I` → **Windows Update** → **Додаткові параметри** → **Активні години** → встанови свій робочий час.

---

## Чому не рекомендується вимикати оновлення назавжди

Оновлення безпеки закривають вразливості — без них система швидко стає вразливою до атак. Краще налаштувати зручний час встановлення ніж вимикати повністю.

Компроміс: вимкни **автоматичне** встановлення але перевіряй і встановлюй оновлення безпеки вручну раз на місяць.
