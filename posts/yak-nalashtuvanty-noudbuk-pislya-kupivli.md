Новий ноутбук з коробки — не готовий до роботи. Виробники встановлюють зайві програми, вимикають важливі функції і залишають системні налаштування на мінімумі. Ось що зробити в перший день.

---

## 1. Оновити Windows одразу

На нових ноутбуках Windows часто місяцями не оновлювалась — є критичні патчі безпеки.

`Параметри` → `Windows Update` → **Перевірити наявність оновлень** → встанови все.

Може знадобитись кілька перезавантажень. Не пропускай — це важливо.

---

## 2. Видалити bloatware виробника

Acer, ASUS, Lenovo, HP, Dell встановлюють десятки непотрібних програм: пробні версії антивірусів, фірмові "оптимізатори", ігрові платформи.

`Параметри` → `Програми` → `Встановлені програми` → видали все що не впізнаєш або не потребуєш.

```powershell
# Масове видалення через PowerShell
Get-AppxPackage | Where-Object { $_.Name -like "*CandyC*" -or $_.Name -like "*BingNews*" } |
  Remove-AppxPackage
```

Детальніше: [Як видалити bloatware з Windows 11](/yak-prybravty-bloatware-windows-11)

---

## 3. Встановити антивірус або перевірити вбудований

Windows Defender вбудований і достатній для більшості задач. Перевір що він увімкнений:

`Безпека Windows` → всі пункти зелені.

Якщо виробник встановив пробний Avast, Norton або McAfee — видали його і залиш Defender.

---

## 4. Налаштувати резервне копіювання

Встанови це до того як щось трапиться.

**OneDrive:** значок в треї → Параметри → Резервне копіювання → увімкни Документи, Зображення, Робочий стіл.

Детальніше: [OneDrive налаштування резервного копіювання](/yak-nalashtuvanty-rezervne-kopiyuvannya-onedrive)

---

## 5. Налаштувати батарею

Новий ноутбук — нова батарея. Щоб прожила довше:

- Встанови **поріг заряду** якщо є програма виробника (Lenovo Vantage, ASUS MyASUS, Dell Power Manager)
- Ідеально: заряджати в діапазоні 20–80%
- Перший раз заряди до 100% і розряди до вимкнення для калібрування

Детальніше: [Налаштування батареї ноутбука](/yak-nalashtuvanty-batareyku-noutbuka-windows)

---

## 6. Перевірити і оновити драйвери

Системні драйвери можуть бути застарілими навіть на новому ноутбуці.

`Win + X` → **Диспетчер пристроїв** → перевір на жовті трикутники.

Для відеокарти — завантаж актуальний драйвер безпосередньо:
- NVIDIA: nvidia.com/drivers
- AMD: amd.com/drivers
- Intel: intel.com/content/www/us/en/download-center

---

## 7. Налаштувати масштаб і роздільну здатність

На нових ноутбуках часто встановлений масштаб 125% або 150% — перевір що підходить тобі.

`Параметри` → `Система` → `Дисплей` → **Масштаб** і **Роздільна здатність**.

---

## 8. Вимкнути непотрібний автозапуск

`Ctrl + Shift + Esc` → **Автозапуск** → вимкни все зайве.

На новому ноутбуці зазвичай 10–15 програм в автозапуску від виробника — більшість не потрібні.

---

## 9. Налаштувати конфіденційність

`Параметри` → `Конфіденційність і безпека` → `Загальне`:
- Вимкни рекламний ідентифікатор
- Вимкни відстеження мови

Детальніше: [Вимкнути шпигунство Windows 11](/windows-11-vymknuty-shpyhunstvo)

---

## 10. Встановити потрібні програми

Список що варто встановити одразу:

```powershell
# Через winget — все одразу
winget install Google.Chrome
winget install Mozilla.Firefox
winget install VideoLAN.VLC
winget install 7zip.7zip
winget install Microsoft.VisualStudioCode
winget install Notepad++.Notepad++
winget install Bitwarden.Bitwarden
```

- Acer: acer.com/register

## Підсумок

Мінімальний чеклист: оновити Windows → видалити bloatware → перевірити Defender → увімкнути OneDrive → оновити драйвери відеокарти. Ці 5 кроків займуть 30–60 хвилин але заощадять години проблем у майбутньому.
