Захаращений робочий стіл уповільнює роботу. Ось як налаштувати його один раз і забути.

---

## Розмір і розташування іконок

Правий клік на робочому столі → **Вигляд**:
- **Великі / Середні / Дрібні значки** — вибери зручний розмір
- **Автоматично впорядковувати значки** — іконки вирівнюються по сітці
- **Вирівнювати значки по сітці** — вмикай якщо потрібен порядок

Прокрути коліщатко миші на робочому столі утримуючи `Ctrl` — плавно змінює розмір іконок.

---

## Системні іконки (Цей ПК, Кошик)

За замовчуванням Windows 11 показує тільки Кошик. Щоб додати інші:

`Win + I` → **Персоналізація** → **Теми** → **Параметри значків робочого столу** → постав галочки на **Комп'ютер**, **Мережа**, **Файли користувача**.

---

## Ярлики для швидкого запуску

Замість того щоб тримати все на робочому столі — закріпи програми на панелі завдань:

Знайди програму в Пуску → правий клік → **Закріпити на панелі завдань**.

Для папок і файлів: правий клік → **Надіслати** → **Робочий стіл (створити ярлик)**.

Щоб відкрити програму гарячою клавішею: правий клік на ярлику → **Властивості** → поле **Швидкий виклик** → натисни комбінацію (наприклад `Ctrl + Alt + N` для Notepad).

---

## Прибрати зайве з контекстного меню Windows 11

Windows 11 додає пункти "Відкрити в Terminal", "Показати більше параметрів" тощо. Якщо заважає — повернути старе меню:

```powershell
# Повернути класичне контекстне меню Windows 10
reg add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve
```

Перезапусти Провідник: `Ctrl + Shift + Esc` → **Провідник Windows** → **Перезапустити**.

Щоб повернути нове меню:
```powershell
reg delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}" /f
```

## Кілька моніторів

Правий клік на робочому столі → **Параметри дисплея** → **Кілька дисплеїв**.

Корисні налаштування:
- **Розширити ці дисплеї** — різний вміст на кожному
- **Дублювати ці дисплеї** — однакове зображення

Перетягни прямокутники моніторів щоб відобразити фізичне розташування.

Панель завдань на всіх моніторах: `Win + I` → **Персоналізація** → **Панель завдань** → **Параметри панелі завдань для кількох дисплеїв** → увімкни **Показувати панель завдань на всіх дисплеях**.

---

## Резюме

Мінімум на робочому столі — максимум продуктивності. Тримай там тільки активні проекти і часто використовувані ярлики. Все інше — в меню Пуск або на панелі завдань. Використовуй `Win + D` щоб миттєво показати/приховати робочий стіл.
