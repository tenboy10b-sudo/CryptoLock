Якщо ПК гальмує, вимикається або вентилятори гудуть на повну — можливо причина в перегріві. Ось як швидко перевірити температуру без складних програм.

---

## Метод 1: Task Manager (Windows 11)

`Ctrl + Shift + Esc` → вкладка **Performance** → вибери **CPU**

У нижній частині показується поточна температура процесора. Доступно тільки в Windows 11 і тільки якщо BIOS передає ці дані.

---

## Метод 2: PowerShell (базово)

```powershell
Get-WmiObject MSAcpi_ThermalZoneTemperature -Namespace "root/wmi" |
  Select-Object @{n='Температура °C';e={($_.CurrentTemperature - 2732) / 10}}
```

Показує температуру теплових зон материнської плати — не завжди відповідає температурі ядер CPU.

---

## Метод 3: HWiNFO64 (найточніший)

Завантаж з [hwinfo.com](https://www.hwinfo.com/download/) → запусти в режимі **Sensors only**.

Показує:
- Температуру кожного ядра CPU окремо
- GPU Core Temp і GPU Memory Temp
- Температуру материнської плати і дисків
- Напругу і швидкість вентиляторів

Запускай у фоні під час гри або навантаження — вгорі таблиці буде колонка **Max** з пиковими значеннями.

---

## Метод 4: Core Temp (тільки CPU)

Завантаж з [alcpu.com/CoreTemp](https://www.alcpu.com/CoreTemp/) — легка утиліта, показує температуру кожного ядра в треї.

При встановленні зніми галочку з додаткового ПЗ.

---

## Нормальні температури

| Компонент | В спокої | Під навантаженням | Критично |
|-----------|----------|-------------------|----------|
| CPU (десктоп) | 30–45°C | 65–80°C | >95°C |
| CPU (ноутбук) | 35–50°C | 75–90°C | >100°C |
| GPU | 30–45°C | 70–85°C | >95°C |
| SSD NVMe | 35–50°C | 55–70°C | >80°C |

Сучасні процесори мають захист — при досягненні критичної температури вони знижують частоту (throttling) або вимикають ПК.

---

## Якщо температура зависока

**Для десктопу:**
- Почисти від пилу — особливо радіатор і вентилятор CPU
- Перевір чи вентилятори обертаються
- Замін термопасту (раз на 3–5 років)
- Перевір потік повітря в корпусі

**Для ноутбука:**
- Почисти вентиляційні отвори
- Поклади на тверду поверхню — не на ліжко чи диван
- Замін термопасту (або зверни в сервіс)
- Використовуй підставку з охолодженням

---

## Резюме

Для швидкої перевірки — Task Manager (Windows 11) або Core Temp. Для детального моніторингу під навантаженням — HWiNFO64. Нормальна температура CPU під навантаженням до 80–85°C для десктопу і до 90°C для ноутбука.
