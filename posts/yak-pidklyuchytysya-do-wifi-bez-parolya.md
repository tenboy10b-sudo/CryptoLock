Підключити новий пристрій до Wi-Fi але не пам'ятаєш пароль? Є кілька способів зробити це швидко.

---

## Спосіб 1: Показати збережений пароль в Windows

Якщо ПК вже підключений до цієї мережі — пароль можна знайти.

```cmd
rem Показати пароль від поточної мережі
netsh wlan show profile name="Назва_мережі" key=clear
```

Шукай рядок **Key Content** — це твій пароль.

```powershell
# Показати паролі від всіх збережених мереж
$profiles = (netsh wlan show profiles) -match "All User Profile" | ForEach-Object { ($_ -split ":")[1].Trim() }
foreach ($profile in $profiles) {
    $pass = (netsh wlan show profile name="$profile" key=clear) -match "Key Content"
    if ($pass) {
        Write-Output "$profile : $(($pass -split ':')[1].Trim())"
    }
}
```

---

## Спосіб 2: WPS кнопка на роутері

Якщо на роутері є кнопка WPS:

1. На новому пристрої вибери мережу → **Підключити через WPS**
2. Протягом 2 хвилин натисни кнопку **WPS** на роутері
3. Пристрій підключиться без пароля

> WPS має відому вразливість. Якщо ним не користуєшся — краще вимкни в налаштуваннях роутера.

---

## Спосіб 3: QR-код на Android

Android (6.0+) дозволяє поділитись Wi-Fi через QR-код:

1. **Налаштування** → **Wi-Fi** → клікни на підключену мережу → **Поділитись** або значок QR
2. З'явиться QR-код — відскануй з іншого телефону або планшета

На Windows можна відсканувати через застосунок **Camera** або **Microsoft Lens**.

---

## Спосіб 4: Показати пароль через налаштування мережі (Windows 11)

`Параметри` → `Мережа і Інтернет` → **Wi-Fi** → **Керування відомими мережами** → вибери мережу → **Переглянути**

Або:

`Параметри` → **Wi-Fi** → клікни на підключену мережу → **Властивості** → **Показати** поруч з паролем.

---

## Спосіб 5: Перевірити на роутері

Якщо нічого не допомогло — зайди в налаштування роутера:

1. Відкрий браузер → `192.168.1.1` (або `192.168.0.1`)
2. Увійди (логін/пароль зазвичай на наклейці знизу роутера)
3. Розділ **Wi-Fi** або **Wireless** → тут пароль у відкритому вигляді

---

## Скинути пароль Wi-Fi (якщо нічого не допомогло)

Якщо зовсім не пам'ятаєш і немає доступу до роутера — тільки скидання роутера:

Утримуй кнопку **Reset** на роутері 10–15 секунд. Роутер скинеться до заводських налаштувань. Новий пароль буде на наклейці знизу.

---

## Підсумок

Найшвидше: `netsh wlan show profile name="Мережа" key=clear` — покаже пароль якщо ПК вже підключений. Для нового пристрою — WPS кнопка або QR-код з Android.
