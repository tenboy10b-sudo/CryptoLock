Повільний інтернет не завжди провина провайдера. Часто проблема у налаштуваннях Windows, DNS або Wi-Fi адаптера. Ось 8 способів які реально допомагають.

---

## Спочатку: виміряй реальну швидкість

Зайди на **speedtest.net** або **fast.com** і запиши результат. Порівняй з тим що написано в твоєму тарифі.

- Якщо швидкість по кабелю значно нижча за тариф — проблема у провайдера або роутера
- Якщо по Wi-Fi повільніше ніж по кабелю — проблема у Wi-Fi
- Якщо однаково повільно і по кабелю і по Wi-Fi — читай далі

---

## 1. Зміни DNS на швидший

DNS за замовчуванням від провайдера часто повільний. Заміна на Google або Cloudflare прискорює завантаження сторінок.

`Win + R` → `ncpa.cpl` → правою кнопкою на активному підключенні → **Властивості** → **IP версії 4** → **Властивості** → **Використовувати наступні адреси DNS-серверів**:

**Cloudflare (найшвидший за тестами):**
```
Основний: 1.1.1.1
Додатковий: 1.0.0.1
```

**Google:**
```
Основний: 8.8.8.8
Додатковий: 8.8.4.4
```

Після зміни очисти DNS кеш:

```cmd
ipconfig /flushdns
```

Детальніше про DNS: [Як налаштувати DNS в Windows](/yak-zminyty-dns-windows)

---

## 2. Вимкни обмеження QoS (зарезервована пропускна здатність)

Windows за замовчуванням резервує до 20% пропускної здатності для системних потреб. Можна вимкнути:

`Win + R` → `gpedit.msc` → **Конфігурація комп'ютера** → **Конфігурація Windows** → **Параметри політики** → **QoS-планувальник пакетів** → **Обмежити резервовану пропускну здатність** → **Enabled** → встав **0%**

> Примітка: це налаштування доступне тільки у Windows Pro і вище.

---

## 3. Вимкни фонові завантаження Windows Update

Windows Update може займати всю смугу у фоні саме коли ти дивишся відео або граєш.

`Параметри` → `Windows Update` → `Додаткові параметри` → `Оптимізація доставки` → **Додаткові параметри** → встанови обмеження:
- **Завантаження у фоні** → 10–20% від смуги
- **Завантаження на передньому плані** → 80–100%

---

## 4. Оновити або перевстановити драйвер мережевого адаптера

Старий або пошкоджений драйвер Wi-Fi або Ethernet — часта причина повільного з'єднання.

`Win + X` → `Диспетчер пристроїв` → **Мережеві адаптери** → клікни правою кнопкою на свій адаптер → **Оновити драйвер**

Якщо не допомогло — видали драйвер і перезавантаж ПК, Windows встановить його заново.

Детальніше: [Як встановити драйвер](/yak-vstanovyty-draiver)

---

## 5. Налаштуй Wi-Fi адаптер на максимальну продуктивність

`Win + X` → `Диспетчер пристроїв` → **Мережеві адаптери** → правою кнопкою на Wi-Fi адаптер → **Властивості** → вкладка **Додатково**:

- **Wireless Mode** → встанови найвище доступне (802.11ac або 802.11ax)
- **Roaming Aggressiveness** → Low (щоб не перемикатись на слабші точки)
- **Transmit Power** → Highest

`Параметри електроживлення` адаптера:
`Win + X` → `Диспетчер пристроїв` → Wi-Fi адаптер → `Властивості` → **Керування живленням** → зніми галочку **"Дозволити вимкнення для економії енергії"**

---

## 6. Перевір і вимкни фонові програми що споживають мережу

`Ctrl + Shift + Esc` → вкладка **Продуктивність** → **Відкрити монітор ресурсів** → вкладка **Мережа**

Побачиш які програми використовують інтернет у фоні. Хмарні сховища (OneDrive, Google Drive, Dropbox) часто синхронізуються і займають смугу.

---

## 7. Переключись на діапазон 5 ГГц

Якщо роутер підтримує 5 ГГц — підключись до нього замість 2.4 ГГц. Швидкість вища, менше перешкод від сусідів.

У списку Wi-Fi мереж знайди назву свого роутера з припискою "5G" або "_5GHz".

**Але:** 5 ГГц має меншу дальність. Якщо ти далеко від роутера — 2.4 ГГц може бути стабільнішим.

---

## 8. Очисти DNS кеш і скинь TCP/IP

Якщо сайти відкриваються повільно або з перебоями:

Відкрий CMD від адміністратора:

```cmd
ipconfig /flushdns
netsh int ip reset
netsh winsock reset
```

Перезавантаж комп'ютер після виконання.

---

## Підсумок

Найбільший ефект зазвичай дають: зміна DNS на Cloudflare, оновлення драйвера адаптера і переключення на 5 ГГц. Решта — додаткове налаштування для максимізації швидкості.
