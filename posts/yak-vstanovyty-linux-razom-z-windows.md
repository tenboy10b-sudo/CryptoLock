Подвійне завантаження (dual boot) дозволяє мати Windows і Linux на одному комп'ютері і вибирати систему при кожному запуску. Без ризику для Windows якщо робити правильно.

---

## Що потрібно перед початком

- USB флешка 8 ГБ+ для інсталятора Linux
- Мінімум 30–50 ГБ вільного місця на диску
- Вимкнений **Fast Startup** і **Secure Boot** (або увімкнений якщо дистрибутив підтримує)
- Резервна копія важливих файлів

---

## Крок 1: Вимкнути Fast Startup

Fast Startup в Windows "заморожує" диск — Linux не зможе його змонтувати.

`Параметри` → `Система` → `Живлення і сон` → `Додаткові параметри живлення` → **Вибрати дії кнопок живлення** → **Змінити параметри** → зніми **Швидкий запуск**.

---

## Крок 2: Звільнити місце на диску для Linux

```powershell
# Переглянути розділи
Get-Partition | Select-Object DriveLetter, Size, Type
```

`Win + X` → **Керування дисками** → правою кнопкою на розділ C → **Стиснути том**:
- Введи обсяг для Linux: 30000–50000 МБ (30–50 ГБ)
- Натисни **Стиснути**

З'явиться "Нерозподілений простір" — Linux встановиться туди.

---

## Крок 3: Завантажити і записати Ubuntu на флешку

1. Завантаж Ubuntu LTS з **ubuntu.com/download**
2. Завантаж **Rufus** з rufus.ie
3. Запусти Rufus → вибери флешку → вибери ISO → **Старт**

---

## Крок 4: Завантажитись з флешки

Перезавантаж ПК → при старті натисни клавішу Boot Menu (F12, F11, Esc або Del залежно від материнської плати).

Вибери USB флешку → завантажиться Ubuntu.

---

## Крок 5: Встановити Ubuntu

У майстрі встановлення:

1. **Тип встановлення** → **Встановити разом з Windows Boot Manager**

Ubuntu автоматично знайде нерозподілений простір і встановиться туди + налаштує GRUB.

> Якщо варіант не з'явився — вибери **Інший варіант** і вручну вкажи нерозподілений простір як `/` (root) формат ext4.

2. Встанови часовий пояс, ім'я і пароль
3. Дочекайся завершення → перезавантаж

---

## Крок 6: Вибір системи при завантаженні

Після встановлення при кожному запуску з'являється **GRUB меню**:
- Ubuntu
- Windows Boot Manager

Вибирай стрілками. За замовчуванням через 10 секунд завантажується перша система.

---

## Змінити систему за замовчуванням

### Зробити Windows за замовчуванням

В Ubuntu:

```bash
sudo nano /etc/default/grub
```

Знайди рядок `GRUB_DEFAULT=0` і заміни `0` на номер Windows у меню (зазвичай 2 або 4, рахуй з 0).

```bash
sudo update-grub
```

### Через графічний інструмент

В Ubuntu: встанови **grub-customizer** → `sudo apt install grub-customizer`

---

## Типові проблеми

### Windows не завантажується після встановлення Linux

GRUB не побачив Windows. В Ubuntu:

```bash
sudo update-grub
```

### GRUB не з'являється — одразу вантажиться Windows

Secure Boot блокує GRUB. Вимкни Secure Boot в BIOS або встанови Ubuntu в режимі UEFI.

### Час відрізняється між Windows і Linux на 2–3 години

Windows і Linux по-різному зберігають час в RTC.

В Ubuntu виправляється командою:
```bash
timedatectl set-local-rtc 1 --adjust-system-clock
```

### Не вистачає місця в Ubuntu

```bash
# Переглянути розмір розділів в Linux
df -h
lsblk
```

---

## Видалити Linux і повернутись до тільки Windows

1. Завантажся в Windows
2. Відкрий **CMD від адміністратора**:
```cmd
bootrec /fixmbr
bootrec /fixboot
```
3. Відкрий **Керування дисками** → видали розділи Linux (типу Linux або без букви)
4. Розшир розділ Windows на звільнений простір

---

## Підсумок

Головне: вимкни Fast Startup → звільни місце через Стиснення тому → встанови Ubuntu в режимі "разом з Windows" → GRUB налаштується автоматично. Якщо Windows зникло з GRUB — `sudo update-grub` в Ubuntu вирішує.
