Налаштувати корпоративну пошту в Windows можна за 5 хвилин якщо знати правильну послідовність дій.

---

## Outlook — підключення Microsoft 365 або Exchange

### Автоматичне налаштування

1. Відкрий Outlook
2. **Файл** → **Додати обліковий запис**
3. Введи корпоративний email → **Підключити**
4. Outlook автоматично знайде налаштування Exchange/M365
5. Введи пароль → підтвердь через 2FA якщо потрібно

Якщо Outlook не знайшов сервер автоматично — запитай у IT-відділу:
- Тип сервера: Exchange, IMAP або POP3
- Адреса сервера і порти

### Ручне налаштування Exchange

`Файл` → **Параметри облікового запису** → **Облікові записи** → **Створити** → **Exchange або Office 365**:
- Ім'я користувача: `firstname.lastname@company.com`
- Пароль: корпоративний пароль
- Сервер: `mail.company.com` або автовиявлення

---

## Mail (вбудований додаток Windows)

Простіший варіант для базової роботи з поштою.

`Пуск` → **Пошта** → **Облікові записи** → **Додати обліковий запис** → **Exchange**:
- Email: корпоративний
- Пароль: корпоративний

Додаток автоматично підключиться до Microsoft 365 або Exchange якщо налаштований Autodiscover.

## Налаштування підпису в Outlook

`Файл` → **Параметри** → **Пошта** → **Підписи** → **Новий**:

```
Іван Петренко
Менеджер проектів | ТОВ «Компанія»
📞 +380 44 123 4567
✉ ivan.petrenko@company.com
🌐 www.company.com
```

Вибери який підпис використовувати для нових листів і відповідей.

---

## Налаштування правил для автоматичного сортування

`Основне` → **Правила** → **Керування правилами і сповіщеннями** → **Нове правило**:

Приклад — автоматично переміщувати листи від клієнта в папку:
1. **Запустити правило для отриманих повідомлень**
2. Умова: **від користувача або списку розсилки** → введи email клієнта
3. Дія: **перемістити в папку** → вибери або створи папку
4. **Готово**

---

## Спільний поштова скринька

Якщо маєш доступ до корпоративної спільної скриньки (`info@company.com`):

`Файл` → **Параметри облікового запису** → вибери акаунт → **Змінити** → **Інші параметри** → **Додатково** → **Додати** → введи адресу спільної скриньки.

## Проблема: пошта не синхронізується

```powershell
# Перевірити підключення до сервера Exchange/M365
Test-NetConnection -ComputerName "outlook.office365.com" -Port 443
Test-NetConnection -ComputerName "mail.company.com" -Port 443
```

Якщо порт закритий — проблема з мережею або брандмауером.

**Очистити кеш Outlook:**

1. Закрий Outlook
2. `Win + R` → `%localappdata%\Microsoft\Outlook`
3. Видали файли `.ost` (оффлайн копія) — Outlook відтворить їх при наступному запуску

## Архівування старих листів

**Автоархів:**
`Файл` → **Параметри** → **Додатково** → **Параметри автоархіву** → налаштуй очищення листів старших 6–12 місяців.

**Вручну:**
`Файл` → **Засоби** → **Очистити старі елементи** → вибери папки і дату.

## Підсумок

Outlook з Microsoft 365 — введи email → Підключити → Outlook знайде все сам. Якщо не знайшов — запитай у IT адресу сервера. Mail (вбудований) — простіша альтернатива для базових потреб. Правила сортування і підпис налаштовуй одразу — заощадить час щодня.
