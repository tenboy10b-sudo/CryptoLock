BIOS (або UEFI на сучасних ПК) — це прошивка яка запускається до Windows. Зайти туди потрібно щоб змінити порядок завантаження, налаштувати Secure Boot, увімкнути віртуалізацію або оновити прошивку.

## Спосіб 1: Клавіша при включенні ПК

Найшвидший спосіб. Відразу при включенні ПК (до появи логотипу Windows) натискай потрібну клавішу кілька разів.

### Клавіші входу по виробниках

| Виробник | Клавіша BIOS |
|---|---|
| ASUS | **Del** або F2 |
| Gigabyte | **Del** |
| MSI | **Del** |
| ASRock | **Del** або F2 |
| Intel NUC | **F2** |
| HP | **F10** або Esc |
| Dell | **F2** |
| Lenovo | **F1** або F2 (ноутбуки: **Fn + F2**) |
| Acer | **Del** або F2 |
| Toshiba | **F2** |
| Samsung | **F2** |
| Sony VAIO | **F2** або Assist |

> Якщо не знаєш виробника — спробуй **Del** або **F2**, це найпоширеніші варіанти.

---

## Спосіб 2: Через Параметри Windows (надійний)

Цей спосіб зручний якщо ПК завантажується занадто швидко і не встигаєш натиснути клавішу.

**Windows 11:**
`Win + I` → **Система** → **Відновлення** → **Розширений запуск** → **Перезавантажити зараз**

**Windows 10:**
`Win + I` → **Оновлення і безпека** → **Відновлення** → **Розширений запуск** → **Перезавантажити зараз**

Після перезавантаження: **Усунення неполадок** → **Додаткові параметри** → **Параметри мікропрограми UEFI** → **Перезавантажити**.

---

## Спосіб 3: Через командний рядок

Відкрий CMD від адміністратора:

```cmd
shutdown /r /fw /t 0
```

ПК перезавантажиться одразу в BIOS.

---

## Спосіб 4: Утримуй Shift при перезавантаженні

В меню Пуск → Живлення → утримуй **Shift** і натисни **Перезавантаження**.

Потрапиш у середовище відновлення → **Усунення неполадок** → **Додаткові параметри** → **Параметри мікропрограми UEFI**.

---

## Швидке завантаження заважає зайти в BIOS?

Якщо ПК завантажується миттєво і клавіша не спрацьовує — проблема в **Fast Boot** або **Fast Startup**.

Вимкни Fast Startup в Windows:
`Win + S` → **Параметри живлення** → **Вибрати дію кнопок живлення** → **Змінити параметри які недоступні зараз** → зніми галочку **Увімкнути швидкий запуск**.

---

## Що робити в BIOS

Основні розділи:

- **Main / Info** — загальна інформація про систему
- **Boot** — порядок завантаження, пріоритет пристроїв
- **Security** — паролі, Secure Boot, TPM
- **Advanced** — налаштування CPU, USB, портів
- **Exit** — збереження (`F10` + Enter) або скидання налаштувань

> Не змінюй те чого не розумієш. Неправильні налаштування можуть призвести до того що ПК не завантажиться.
