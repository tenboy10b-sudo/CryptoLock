Bluetooth підключення іноді капризує — пристрій не знаходиться, з'єднання рветься або звук пропадає. Ось як правильно підключити і вирішити типові проблеми.

---

## Підключення нового пристрою

1. Переведи пристрій у режим сполучення (зазвичай утримуй кнопку живлення 3–5 секунд поки не заблимає індикатор)
2. На Windows: `Параметри` → `Bluetooth і пристрої` → **Додати пристрій** → **Bluetooth**
3. Вибери пристрій зі списку → **Підключити**

Або через системний трей: клікни на значок мережі → Bluetooth → **Додати пристрій**.

---

## Якщо Bluetooth не вмикається

```powershell
# Перевірити статус служби Bluetooth
Get-Service -Name bthserv | Select-Object Status, StartType

# Запустити службу якщо зупинена
Start-Service -Name bthserv
Set-Service -Name bthserv -StartupType Automatic

# Перевірити чи є адаптер
Get-PnpDevice | Where-Object { $_.Class -eq "Bluetooth" }
```

Якщо адаптера немає — перевір Диспетчер пристроїв:

`Win + X` → **Диспетчер пристроїв** → **Bluetooth** — чи є пристрій, чи є жовтий трикутник.

Якщо адаптер є але з трикутником → оновити драйвер: [Як встановити драйвер](/yak-vstanovyty-draiver)

## Пристрій не знаходиться при пошуку

- Переконайся що пристрій в режимі сполучення (блимає індикатор)
- Піднеси ближче — Bluetooth діє до 10 метрів без перешкод
- Вимкни і увімкни Bluetooth на ПК
- Перезавантаж пристрій

```powershell
# Перезапустити Bluetooth адаптер
Get-PnpDevice | Where-Object { $_.Class -eq "Bluetooth" -and $_.Status -eq "OK" } |
  Disable-PnpDevice -Confirm:$false
Start-Sleep 3
Get-PnpDevice | Where-Object { $_.Class -eq "Bluetooth" } |
  Enable-PnpDevice -Confirm:$false
```

## Налаштування звуку для Bluetooth навушників

Після підключення навушників перевір що вони вибрані як пристрій відтворення:

Правою кнопкою на значок звуку → **Параметри звуку** → **Виведення** → вибери навушники.

**Проблема з мікрофоном:** Bluetooth навушники часто мають два режими — **Stereo** (якісний звук, без мікрофону) і **Hands-Free** (гірший звук, але мікрофон працює). Перемикається автоматично при відкритті додатку що використовує мікрофон.

## З'єднання рветься або звук заїкається

- Прибери перешкоди між ПК і пристроєм (стіни, металеві предмети)
- Вимкни Wi-Fi на 2.4 ГГц — він конфліктує з Bluetooth (обидва на 2.4 ГГц)
- Оновити драйвер Bluetooth адаптера
- Вимкни **економію енергії** для Bluetooth адаптера:

`Диспетчер пристроїв` → **Bluetooth** → адаптер → **Властивості** → **Керування живленням** → зніми **"Дозволити вимкнення для економії енергії"**

---

## Видалити пристрій і підключити заново

Іноді допомагає "забути" пристрій і додати знову:

`Параметри` → **Bluetooth і пристрої** → клікни `...` поруч з пристроєм → **Видалити пристрій** → потім підключи заново.

---

## Підсумок

`Параметри` → **Bluetooth і пристрої** → **Додати пристрій** — стандартне підключення. Не знаходиться — перезапусти Bluetooth через PowerShell або Диспетчер пристроїв. Заїкається звук — вимкни економію енергії адаптера і перевір чи не конфліктує з Wi-Fi 2.4 ГГц.
