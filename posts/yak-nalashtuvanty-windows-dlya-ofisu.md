Офісний ПК має особливі вимоги: стабільність, безпека корпоративних даних і відсутність відволікань. Ось стандартний набір налаштувань для робочого місця.

---

## 1. Вимкнути ігри і розваги

`Параметри` → `Програми` → `Встановлені програми` → видали:
- Microsoft Solitaire Collection
- Xbox і Xbox Game Bar (якщо не потрібно)
- Tips (Поради)

```powershell
Get-AppxPackage -AllUsers *MicrosoftSolitaireCollection* | Remove-AppxPackage -AllUsers
Get-AppxPackage -AllUsers *XboxApp* | Remove-AppxPackage -AllUsers
Get-AppxPackage -AllUsers *XboxGamingOverlay* | Remove-AppxPackage -AllUsers
```

---

## 2. Вимкнути сповіщення і відволікання

`Параметри` → `Система` → `Сповіщення`:
- Вимкни сповіщення від непотрібних програм
- Вимкни **"Показувати поради про Windows"**

`Параметри` → `Персоналізація` → `Пуск`:
- Вимкни **"Показувати рекомендації"**

---

## 3. Налаштувати автоматичне блокування

`Параметри` → `Облікові записи` → `Параметри входу` → **Якщо ви були відсутні** → **Коли ПК виходить зі сну**.

Або через заставку з паролем:
`Win + R` → `desk.cpl` → **Заставка** → встанови 10 хвилин → **Захистити паролем**.

---

## 4. Шифрування диска BitLocker

Для корпоративних даних шифрування обов'язкове — якщо ноутбук загублений, дані недоступні.

`Пуск` → **Керування BitLocker** → **Увімкнути BitLocker**

Детальніше: [Як увімкнути BitLocker](/yak-uvimknuty-bitlocker-windows-11)

---

## 5. Налаштувати корпоративний VPN

Для доступу до внутрішніх ресурсів поза офісом:

`Параметри` → `Мережа і Інтернет` → **VPN** → **Додати VPN**

Або через клієнт корпоративного VPN (Cisco AnyConnect, GlobalProtect тощо) — встанови і налаштуй відповідно до інструкції IT-відділу.

---

## 6. Підключитись до корпоративного принтера

Через мережу або Print Server: [Мережевий принтер у домені](/yak-nalashtuvanty-printery-merezhevy-domen)

---

## 7. Налаштувати корпоративну пошту

**Outlook:** Файл → Додати обліковий запис → введи корпоративний email → Підключити.

При підключенні до Exchange або Microsoft 365 — Outlook знайде налаштування автоматично.

---

## 8. Налаштувати OneDrive для бізнесу

OneDrive для бізнесу (SharePoint) відрізняється від особистого.

Клікни на значок OneDrive → увійди з **корпоративним** акаунтом (не особистим Microsoft) → папки з'являться в Провіднику.

---

## 9. Стандартні програми для офісу

```powershell
# Встановити через winget
winget install Microsoft.Office         # Microsoft 365 (потрібна ліцензія)
winget install 7zip.7zip                # архіватор
winget install Adobe.Acrobat.Reader.64-bit  # PDF
winget install VideoLAN.VLC             # відеоплеєр
winget install Zoom.Zoom                # відеоконференції
winget install Microsoft.Teams          # Teams
```

---

## 10. Вимкнути USB-носії (якщо потрібно)

Якщо корпоративна політика забороняє флешки:

```powershell
# Заблокувати всі USB носії (тільки читання)
Set-ItemProperty "HKLM:\SYSTEM\CurrentControlSet\Services\USBSTOR" `
  -Name "Start" -Value 4 -Type DWord
```

Або через GPO: [Заборона USB через GPO](/grupova-polityka-zaborona-usb)

---

## 11. Налаштувати резервне копіювання

Корпоративні документи мають резервуватись регулярно.

Якщо є корпоративний SharePoint або OneDrive — синхронізація автоматична.

Для локального бекапу: [Резервне копіювання Windows](/rezervne-kopiyuvannya-windows)

---

## 12. Встановити корпоративний антивірус або перевірити Defender

Якщо компанія використовує Defender for Endpoint або ESET/Kaspersky — переконайся що він встановлений і оновлений.

Статус: `Безпека Windows` → всі розділи зелені.

---

## Підсумок

Мінімальний стандарт для корпоративного ПК: BitLocker увімкнений → автоблокування через 10 хвилин → корпоративний VPN налаштований → OneDrive для бізнесу підключений → антивірус активний. Ці 5 пунктів захищають і дані компанії і репутацію при втраті або крадіжці пристрою.
