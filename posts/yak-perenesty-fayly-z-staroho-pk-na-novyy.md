Купив новий ПК — тепер треба перенести все зі старого. Залежно від обсягу даних і ситуації є кілька способів від простих до просунутих.

---

## Що переносити

**Обов'язково:**
- Документи, фото, відео, музика
- Закладки браузера
- Паролі (через менеджер паролів або браузер)
- Ліцензійні ключі програм

**За потреби:**
- Налаштування Windows
- Профілі програм (Outlook, Thunderbird тощо)
- Ігрові збереження

**Не переносити:**
- Системні файли Windows
- Встановлені програми (краще встановити заново)

---

## Спосіб 1: Зовнішній диск або USB (найпростіший)

1. Підключи зовнішній диск до старого ПК
2. Скопіюй папки: `Документи`, `Завантаження`, `Зображення`, `Відео`, `Музика`, Робочий стіл
3. Підключи диск до нового ПК
4. Скопіюй файли на місце

```powershell
# Копіювати всі папки користувача на зовнішній диск E:
$userFolders = @("Documents","Downloads","Pictures","Videos","Music","Desktop")
foreach ($folder in $userFolders) {
    $src = "$env:USERPROFILE\$folder"
    $dst = "E:\Backup\$folder"
    if (Test-Path $src) {
        Copy-Item -Path $src -Destination $dst -Recurse -Force
        Write-Output "✓ $folder скопійовано"
    }
}
```

## Спосіб 2: Через локальну мережу

Якщо обидва ПК в одній Wi-Fi мережі — швидко і без дисків.

**На старому ПК — відкрити спільний доступ до папки:**

Правою кнопкою на папку → **Надати доступ** → **Конкретні користувачі** → **Всі** → **Читання і запис** → **Поділитись**

**На новому ПК — підключитись:**

`Win + R` → `\\IP-старого-ПК` (наприклад `\\192.168.1.100`) → скопіюй файли.

Або через мережевий диск: [Як підключити мережевий диск](/yak-pidklyuchyty-merezhevyy-dysk-windows)

---

## Спосіб 3: OneDrive або Google Drive

Найзручніший якщо файлів не дуже багато (до 15 ГБ безкоштовно в OneDrive).

**На старому ПК:**
1. Перемісти всі важливі папки в `C:\Users\ІМ'Я\OneDrive`
2. Зачекай поки синхронізується

**На новому ПК:**
1. Увійди в той самий Microsoft або Google акаунт
2. Файли завантажаться автоматично

---

## Спосіб 4: Кабель USB-USB або Ethernet

Для великих обсягів даних (100+ ГБ) — найшвидший варіант.

**USB-USB:** потрібен спеціальний кабель "Easy Transfer Cable" або "PC-to-PC" — є на Amazon. Після підключення запускається майстер передачі даних.

**Ethernet напряму:** з'єднай ПК кабелем → на обох встанови статичний IP в одній підмережі → передавай через спільні папки або Robocopy.

---

## Спосіб 5: Robocopy — для великих обсягів

```powershell
# Синхронізувати папки через мережу зі збереженням прав і метаданих
robocopy "\\192.168.1.100\Users\OldUser" "C:\Users\NewUser" `
  /MIR /COPYALL /R:3 /W:5 /LOG:"C:\transfer.log"

# /MIR — дзеркальне копіювання
# /COPYALL — копіювати всі атрибути
# /R:3 — 3 спроби при помилці
# /W:5 — чекати 5 секунд між спробами
# /LOG — зберегти лог
```

## Перенести закладки браузера

**Chrome/Edge:** Налаштування → Акаунт → увійди в Google/Microsoft акаунт → закладки синхронізуються автоматично.

**Firefox:** Закладки → Показати всі → Імпорт/резервне копіювання → Експортувати у HTML → перенеси файл.

---

## Перенести паролі

**Через браузер:** якщо збережені в Chrome або Edge — увійди в акаунт на новому ПК.

**Через менеджер паролів:** Bitwarden, KeePass — просто увійди в акаунт або перенеси файл бази.

**Вручну:** Credential Manager → `Win + R` → `control keymgr.dll` → можна експортувати.

---

## Знайти ліцензійні ключі програм

```powershell
# Ключ продукту Windows
(Get-WmiObject -query 'select * from SoftwareLicensingService').OA3xOriginalProductKey

# Ключі встановленого ПЗ (шукає в реєстрі)
Get-ChildItem "HKLM:\SOFTWARE\Microsoft\Windows NT\CurrentVersion\SoftwareProtectionPlatform"
```

Для офісного ПЗ — перевір email з підтвердженням покупки або сайт виробника за логіном.

---

## Підсумок

Для більшості: зовнішній диск + ручне копіювання папок користувача. Для зручності — OneDrive синхронізація (налаштуй заздалегідь). Для великих обсягів — Robocopy через мережу. Програми встановлюй заново — не намагайся переносити `Program Files`.
