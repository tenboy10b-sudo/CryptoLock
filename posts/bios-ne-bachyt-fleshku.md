Найпоширеніша ситуація: треба завантажитись з флешки, але вона не з'являється в списку Boot Menu. Причин може бути кілька — розберемо кожну по порядку.

## Причина 1: Secure Boot увімкнений

Secure Boot блокує завантаження з носіїв які не мають цифрового підпису Microsoft. Більшість Linux дистрибутивів і деякі утиліти через це не видно в меню.

**Вирішення:**

1. Зайди в BIOS/UEFI (при старті ПК натискай `Del`, `F2` або `F10` — залежно від материнської плати)
2. Знайди розділ **Security** або **Boot**
3. Знайди **Secure Boot** → встанови **Disabled**
4. Збережи та перезавантажся (зазвичай `F10`)

---

## Причина 2: Увімкнений режим UEFI (флешка записана під Legacy)

Якщо флешка записана в режимі MBR/Legacy але BIOS налаштований тільки на UEFI — вона не відобразиться.

**Вирішення:**

1. У BIOS знайди **Boot Mode** або **CSM (Compatibility Support Module)**
2. Зміни на **Legacy + UEFI** або увімкни **CSM Support**
3. Збережи та перезавантажся

Або перезапиши флешку в режимі GPT/UEFI через Rufus:
- Відкрий Rufus → вибери флешку → Схема розділів: **GPT** → Цільова система: **UEFI**

---

## Причина 3: Флешка не розпізнається фізично

**Перевір:**
- Спробуй інший USB-порт (бажано USB 2.0, задня панель ПК)
- Перевір флешку на іншому комп'ютері
- Спробуй іншу флешку

Деякі старі BIOS не підтримують USB 3.0 при завантаженні — використовуй USB 2.0 порт (зазвичай чорного кольору, синій = USB 3.0).

---

## Причина 4: Fast Boot увімкнений

Fast Boot пропускає частину перевірок при старті і може ігнорувати USB-пристрої.

**Вирішення:**

1. У BIOS знайди **Fast Boot** → встанови **Disabled**
2. Або в Windows: `Win + I` → Система → Живлення та сон → Додаткові параметри живлення → Дії кнопок живлення → вимкни **Швидкий запуск**

---

## Причина 5: Флешка записана неправильно

Просто скопіювати ISO на флешку недостатньо — треба записати правильно.

**Правильний спосіб — через Rufus:**

1. Завантаж Rufus з rufus.ie
2. Вставте флешку (мінімум 8 ГБ)
3. Вибери ISO-образ
4. Натисни **Старт**

---

## Причина 6: USB-завантаження вимкнене в BIOS

**Вирішення:**

1. У BIOS знайди розділ **Advanced** або **USB Configuration**
2. Знайди **USB Boot Support** або **Legacy USB Support** → встанови **Enabled**

---

## Причина 7: Неправильний порядок завантаження

Флешка є в списку але система завантажується з диска.

**Вирішення:**

1. У BIOS перейди у **Boot** → **Boot Priority** або **Boot Order**
2. Перетягни флешку на перше місце або натисни `+` щоб підняти вгору
3. Збережи (`F10`) і перезавантажся

Або при старті натисни **F8**, **F11** або **F12** (залежно від материнської плати) щоб відкрити одноразове Boot Menu і вибрати флешку звідти.

---

## Поширені клавіші входу в BIOS/Boot Menu

| Виробник | BIOS | Boot Menu |
|---|---|---|
| ASUS | Del, F2 | F8 |
| Gigabyte | Del | F12 |
| MSI | Del | F11 |
| ASRock | Del, F2 | F11 |
| HP | F10, Esc | F9 |
| Dell | F2 | F12 |
| Lenovo | F1, F2 | F12 |
