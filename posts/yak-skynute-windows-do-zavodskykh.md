Якщо Windows працює дуже погано, є вірус якого не можна видалити, або хочеш передати комп'ютер іншій людині — скидання до заводських налаштувань вирішує проблему без флешки і диска.

## Два варіанти скидання

| Варіант | Що відбувається | Коли використовувати |
|---|---|---|
| **Зберегти мої файли** | Видаляє програми і налаштування, файли в Documents, Desktop, Downloads залишаються | Система гальмує, є баги |
| **Видалити все** | Повне очищення — як новий ПК | Продаж, вірус, серйозні проблеми |

---

## Як запустити скидання

### З робочого Windows (найпростіший варіант)

**Windows 11:**
`Win + I` → **Система** → **Відновлення** → **Скинути цей ПК** → **Скинути ПК**

**Windows 10:**
`Win + I` → **Оновлення і безпека** → **Відновлення** → **Скинути цей ПК** → **Почати**

### З екрану входу (якщо Windows не завантажується нормально)

На екрані входу натисни **Shift** і клікни **Живлення → Перезавантаження**.

Відкриється середовище відновлення → **Усунення неполадок** → **Скинути цей ПК**.

### Через налаштування завантаження

При старті натискай `F11` (або `F8` на старіших ПК) → **Усунення неполадок** → **Скинути цей ПК**.

## Покроково: скидання з робочого Windows

**Крок 1.** Відкрий Параметри як вказано вище → натисни **Скинути ПК**.

**Крок 2.** Вибери варіант:
- **Зберегти мої файли** → продовж з кроку 3
- **Видалити все** → продовж з кроку 3

**Крок 3.** Вибери спосіб перевстановлення Windows:
- **Хмарне завантаження** — скачає нову копію Windows з інтернету (~4 ГБ). Рекомендую якщо є хороший інтернет.
- **Локальне перевстановлення** — використовує файли які вже є на диску. Швидше але якщо система пошкоджена — може не спрацювати.

**Крок 4.** Якщо вибрав "Видалити все" — побачиш ще один вибір:
- **Просто видалити файли** — для особистого використання, швидше
- **Видалити диск повністю** — для передачі іншій людині, безпечніше (займає кілька годин)

**Крок 5.** Натисни **Скинути** — процес почнеться автоматично.

**Крок 6.** ПК перезавантажиться кілька разів. Не вимикай. Процес займає від 30 хвилин до 2 годин.

## Що відбувається з файлами при варіанті "Зберегти мої файли"

✅ Залишається:
- Файли в папках Документи, Завантаження, Робочий стіл, Зображення, Відео, Музика
- Налаштування облікового запису Microsoft

❌ Видаляється:
- Всі встановлені програми
- Налаштування Windows
- Програми з Microsoft Store (але можна перевстановити)

## Важливо перед скиданням

1. **Зроби резервну копію** важливих файлів на зовнішній диск або хмару — навіть при варіанті "Зберегти файли" краще підстрахуватись
2. **Запиши паролі** від програм які потрібно перевстановити
3. **Знайди ліцензійні ключі** для платних програм
4. **Переконайся що є живлення** — не роби скидання від батареї на ноутбуку

---

## Якщо скидання не допомогло

Якщо після скидання проблеми залишились — швидше за все проблема апаратна (диск, RAM) а не програмна. Зроби:
```cmd
chkdsk C: /f /r
```
І перевір RAM утилітою Windows Memory Diagnostic: `Win + S` → **Діагностика пам'яті Windows**.
