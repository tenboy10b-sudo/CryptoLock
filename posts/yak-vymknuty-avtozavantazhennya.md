Чим більше програм запускається разом з Windows — тим повільніше завантажується система. Після встановлення різного ПЗ в автозавантаженні накопичується десятки непотрібних програм. Розберемо як це виправити.

## Спосіб 1: Через Диспетчер завдань (найпростіший)

**Крок 1.** Натисни `Ctrl + Shift + Esc` — відкриється Диспетчер завдань.

**Крок 2.** Перейди на вкладку **Автозагрузка** (або **Автозапуск** в Windows 11).

**Крок 3.** Побачиш список програм. У колонці **Вплив на запуск** звертай увагу на позначки:
- **Висока** — суттєво гальмує завантаження
- **Середня** — помірний вплив
- **Низька** — майже не впливає

**Крок 4.** Натисни правою кнопкою на програму яку хочеш прибрати → **Вимкнути**.

### Що можна сміливо вимикати

- Торрент-клієнти (uTorrent, qBittorrent)
- Месенджери (Telegram, Discord, Skype) — запустиш вручну коли потрібно
- Програми оновлення (Adobe Updater, Spotify)
- Ігрові клієнти (Steam, Epic Games)
- OneDrive — якщо не користуєшся

### Що НЕ варто вимикати

- Антивірус
- Драйвери звукової карти, відеокарти
- `Windows Security` і системні процеси

---

## Спосіб 2: Через msconfig

**Крок 1.** `Win + R` → введи `msconfig` → Enter.

**Крок 2.** Вкладка **Автозагрузка** → тут посилання на Диспетчер завдань в Windows 10/11.

**Крок 3.** На вкладці **Загальні** можна вибрати **Вибірковий запуск** і зняти галочку **Завантажувати елементи автозапуску** — але це агресивний метод, краще вимикати вибірково.

---

## Спосіб 3: Через реєстр (для досвідчених)

Програми автозапуску зберігаються в реєстрі за адресами:

```
HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
```

**Крок 1.** `Win + R` → `regedit` → Enter.

**Крок 2.** Перейди по одному з шляхів вище.

**Крок 3.** Праворуч побачиш список програм. Натисни правою кнопкою на непотрібну → **Видалити**.

> **Увага:** перед редагуванням реєстру зроби резервну копію: Файл → Експорт.

---

## Результат

Після вимкнення зайвих програм Windows завантажується на 30–60% швидше. Якщо програма потрібна — просто запусти її вручну після старту системи.
