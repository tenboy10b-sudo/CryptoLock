Неправильний або застарілий драйвер — причина більшості апаратних проблем: немає звуку, глючить відеокарта, не працює Wi-Fi. Розберемо як зробити все правильно.

## Що таке драйвер і навіщо він потрібен

Драйвер — це програма-посередник між Windows і залізом. Без драйвера Windows не "знає" як спілкуватись з відеокартою, звуковою картою чи Wi-Fi адаптером.

---

## Крок 1. Дізнайся що потрібно оновити

`Win + X` → **Диспетчер пристроїв**.

Пристрої з проблемами позначені:
- 🟡 Жовтий знак оклику — драйвер є але є проблема
- ❓ Знак питання — пристрій не розпізнано, драйвера нема
- ⬇️ Стрілка вниз — пристрій вимкнений

Клікни правою кнопкою на пристрій → **Властивості** → вкладка **Відомості** → **ІД обладнання** — отримаєш точний ідентифікатор для пошуку драйвера.

---

## Крок 2. Де шукати драйвер

### Варіант А: Сайт виробника (найкраще)

| Тип пристрою | Де шукати |
|---|---|
| Ноутбук | Сайт виробника ноутбука (Lenovo, ASUS, HP, Dell, Acer) |
| Відеокарта NVIDIA | nvidia.com/drivers |
| Відеокарта AMD | amd.com/support |
| Відеокарта Intel | intel.com/content/www/us/en/support |
| Материнська плата | Сайт виробника плати (ASUS, MSI, Gigabyte, ASRock) |
| Мережева карта Realtek | realtek.com |
| Мережева карта Intel | intel.com |

### Варіант Б: Через Windows Update

`Win + I` → **Windows Update** → **Додаткові параметри** → **Додаткові оновлення** → **Оновлення драйверів**.

Підходить для базових драйверів але часто дає не найновіші версії.

### Варіант В: За ІД обладнання

Скопіюй ІД обладнання з Диспетчера пристроїв → встав у пошук на **devid.info** або Google — знайдеш точний драйвер.

---

## Крок 3. Як встановити

### Стандартний інсталятор (.exe)

Просто запусти файл і слідуй інструкціям. Перезавантаж після встановлення.

### Пакет драйвера (.inf файл)

**Крок 1.** Розпакуй архів.

**Крок 2.** Диспетчер пристроїв → правий клік на пристрій → **Оновити драйвер**.

**Крок 3.** → **Пошук драйверів на цьому комп'ютері** → вкажи папку з розпакованим драйвером → Далі.

### Через командний рядок

```cmd
pnputil /add-driver "C:\Drivers\driver.inf" /install
```

---

## Крок 4. Перевір що все працює

Після встановлення і перезавантаження:
- Відкрий Диспетчер пристроїв — жовтих знаків не має бути
- Перевір пристрій в роботі (звук, зображення, мережа)

---

## Якщо після оновлення все стало гірше

### Відкат до попереднього драйвера

Диспетчер пристроїв → правий клік → **Властивості** → вкладка **Драйвер** → **Відкат**.

Якщо кнопка неактивна — попередньої версії немає в системі.

### Видалення проблемного драйвера

Диспетчер пристроїв → правий клік → **Видалити пристрій** → постав галочку **Видалити програмне забезпечення драйвера** → Видалити.

Перезавантаж — Windows встановить базовий драйвер автоматично.

---

## Порада: зроби точку відновлення перед оновленням

`Win + S` → **Створити точку відновлення** → **Створити** → введи назву.

Якщо після оновлення драйвера щось зламається — відкатишся за 2 хвилини.

---

## Як дізнатись версію встановленого драйвера

Диспетчер пристроїв → правий клік → Властивості → вкладка **Драйвер** → рядок **Версія драйвера**.

Або через PowerShell:

```powershell
Get-WmiObject Win32_PnPSignedDriver | Select-Object DeviceName, DriverVersion | Sort-Object DeviceName
```
