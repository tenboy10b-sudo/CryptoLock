Власний обліковий запис для дитини з правильними обмеженнями — краще ніж забороняти комп'ютер взагалі. Windows має вбудований батьківський контроль через Microsoft Family Safety.

---

## Крок 1: Створити дитячий акаунт

1. Зайди на **family.microsoft.com** → **Додати члена сім'ї**
2. Вибери **Дитина** → введи email дитини (або створи новий)
3. Дитина отримає запрошення — прийміть його

Або через Windows:
`Параметри` → `Облікові записи` → `Сім'я та інші користувачі` → **Додати члена сім'ї** → **Дитина**

---

## Крок 2: Обмеження часу екрана

**family.microsoft.com** → вибери дитину → **Час використання екрана**:

- Встанови розклад для кожного дня
- Приклад: будні 14:00–20:00, вихідні 10:00–21:00
- Максимальний час на день: 2–3 години

Дитина отримує сповіщення за 15 і 5 хвилин до закінчення часу.

---

## Крок 3: Фільтрація сайтів

**family.microsoft.com** → **Фільтр вмісту** → **Безпечний пошук**:

- Увімкни **Фільтр вмісту** — блокує дорослий контент
- Увімкни **Безпечний пошук** — фільтрує результати пошуку
- **Тільки дозволені сайти** — дитина може відвідувати тільки сайти зі списку

Додай в білий список: school.edu, youtube.com (якщо дозволяєш), wikipedia.org тощо.

> Фільтрація працює в Microsoft Edge. У Chrome або Firefox — заблокуй їх через обмеження програм.

---

## Крок 4: Обмеження програм і ігор

**family.microsoft.com** → **Програми та ігри**:

- Вікове обмеження: 6+, 12+, 16+, 18+
- Конкретні програми можна заблокувати або дозволити окремо

Щоб дитина не встановлювала нові програми:
`Параметри` → `Програми` → `Розширені налаштування програм` → **Вибір місця для встановлення** → тільки Store або запит дозволу.

---

## Крок 5: Звіти про активність

**family.microsoft.com** → **Активність** — тут видно:
- Які сайти відвідувала дитина
- Скільки часу в яких програмах
- Спроби зайти на заблоковані сайти

Настрой щотижневий звіт на email.

---

## Крок 6: Витрати і покупки

**family.microsoft.com** → **Витрати** → вимкни **Купівлі без схвалення**

Дитина не зможе купити гру або додаток без твого підтвердження через email або телефон.

---

## Локальний батьківський контроль (без Microsoft акаунту)

Якщо не хочеш Microsoft акаунт:

### Заблокувати сайти через hosts файл

```powershell
# Додати заблоковані сайти в hosts
$blocked = @("tiktok.com", "www.tiktok.com", "youtube.com", "www.youtube.com")
$hostsPath = "C:\Windows\System32\drivers\etc\hosts"

foreach ($site in $blocked) {
    Add-Content $hostsPath "127.0.0.1 $site"
}
```

### Обмежити час входу

```cmd
rem Дозволити вхід тільки в певний час
net user ДитячийАкаунт /times:M-F,14:00-20:00;Sa-Su,10:00-21:00
```

### DNS фільтрація (для всієї мережі)

Встанови на роутері DNS з фільтрацією:
- **Cloudflare for Families**: `1.1.1.3` і `1.0.0.3` — блокує дорослий контент
- **OpenDNS Family Shield**: `208.67.222.123` і `208.67.220.123`

## Поради

**Не роби таємно** — поговори з дитиною про обмеження і причини. Таємний контроль підриває довіру.

**Не блокуй все** — дитина знайде обхідний шлях. Краще довіряти але перевіряти через звіти.

**Пояснюй а не забороняй** — чому певні сайти небезпечні важливіше ніж просто заборона.

---

## Підсумок

Microsoft Family Safety — найпростіше рішення для Windows: обмеження часу, фільтрація сайтів і звіти в одному місці. Налаштовується через family.microsoft.com за 15 хвилин. Для мережевого захисту всіх пристроїв — DNS фільтрація на роутері.
