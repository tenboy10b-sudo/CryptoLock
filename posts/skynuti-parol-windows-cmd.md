## Що потрібно знати перед початком

Цей метод працює якщо у тебе є **доступ до облікового запису адміністратора** або є завантажувальний носій Windows.

Існує два основних способи:

1. Через обліковий запис адміністратора (якщо є доступ)
2. Через завантажувальне середовище (якщо немає доступу)

---

## Спосіб 1: через обліковий запис адміністратора

### Крок 1. Відкрий CMD з правами адміністратора

Натисни `Win + X` → вибери **Термінал (Адміністратор)** або **Командний рядок (Адміністратор)**.

### Крок 2. Переглянь список користувачів

```cmd
net user
```

### Крок 3. Скинь пароль

```cmd
net user ІМ'Я_КОРИСТУВАЧА НовийПароль123
```

Наприклад:

```cmd
net user Олег NewPass2025!
```

Якщо все вірно — побачиш `Команда виконана успішно`.

---

## Спосіб 2: через завантажувальне середовище

> Використовуй тільки якщо ти власник комп'ютера і не маєш іншого доступу.

### Крок 1. Завантажся з USB-носія Windows

Вибери **Відновлення системи** → **Усунення несправностей** → **Командний рядок**.

### Крок 2. Заміни utilman.exe на cmd.exe

```cmd
copy c:\windows\system32\utilman.exe c:\windows\system32\utilman.exe.bak
copy c:\windows\system32\cmd.exe c:\windows\system32\utilman.exe
```

### Крок 3. Перезавантаж і скинь пароль

На екрані входу натисни кнопку **Спеціальні можливості** — відкриється CMD. Далі:

```cmd
net user ІМ'Я НовийПароль123
```

### Крок 4. Відновити оригінальний файл (важливо!)

```cmd
copy c:\windows\system32\utilman.exe.bak c:\windows\system32\utilman.exe
```

## Підсумок

| Спосіб | Коли підходить |
|--------|---------------|
| Через адміністратора | Є доступ до іншого облікового запису |
| Через USB | Немає жодного доступу до системи |

> **Порада безпеки:** після відновлення доступу увімкни PIN-код замість пароля — він зручніший і безпечніший.
