Другий монітор підвищує продуктивність на 20–40% за даними досліджень. Підключити і налаштувати його в Windows — справа 5 хвилин.

---

## Підключення

Підключи монітор кабелем до ПК або ноутбука:
- **HDMI** — найпоширеніший, підтримує до 4K 60Hz
- **DisplayPort** — краще для ігор, підтримує 144Hz і вище
- **USB-C / Thunderbolt** — для сучасних ноутбуків
- **VGA** — старий стандарт, тільки для старих моніторів

Якщо на ноутбуці немає потрібного роз'єму — використай USB-адаптер або USB-C хаб.

---

## Вибір режиму відображення

Після підключення натисни `Win + P` — з'явиться панель вибору режиму:

- **Тільки екран ПК** — другий монітор вимкнений
- **Дублювати** — обидва показують однакове зображення (для презентацій)
- **Розширити** — два незалежних робочих столи (для продуктивності)
- **Тільки другий екран** — основний вимкнений

Для щоденної роботи вибирай **Розширити**.

---

## Налаштування розташування моніторів

`Пуск` → `Параметри` → `Система` → `Дисплей`

Вгорі показані прямокутники — твої монітори. Перетягни їх відповідно до фізичного розташування на столі (наприклад, якщо другий монітор стоїть зліва — перетягни прямокутник 2 лівіше прямокутника 1).

Натисни **Визначити** щоб побачити номери моніторів на екранах.

---

## Налаштування роздільної здатності і частоти

Кожен монітор налаштовується окремо.

Клікни на потрібний прямокутник монітора → прокрути вниз:

- **Роздільна здатність** — вибери рекомендовану (зазвичай максимальна нативна)
- **Частота оновлення** — для ігрових моніторів встанови максимальну (144, 165, 240 Hz)

```
Параметри → Система → Дисплей → Додаткові параметри дисплея
→ вибери монітор → Властивості адаптера дисплея
→ вкладка Монітор → Частота оновлення екрана
```

---

## Масштаб для кожного монітора

Якщо монітори різного розміру або роздільної здатності — масштаб може відрізнятись.

`Параметри` → `Система` → `Дисплей` → клікни на монітор → **Масштаб**

Типові значення:
- Full HD (1920×1080) 24" → 100%
- Full HD 27" → 125%
- 4K 27" → 150–200%

---

## Вибір основного монітора

Основний монітор — де з'являється панель завдань і нові вікна за замовчуванням.

`Параметри` → `Дисплей` → клікни на потрібний монітор → постав галочку **Зробити основним дисплеєм**

---

## Типові проблеми

### Другий монітор не визначається

1. Перевір кабель — від'єднай і під'єднай знову
2. `Параметри` → `Дисплей` → натисни **Виявити**
3. Оновити драйвер відеокарти: [Як встановити драйвер](/yak-vstanovyty-draiver)
4. Спробуй інший кабель або роз'єм

### Розмитий або нечіткий текст на другому моніторі

Проблема масштабу. Для конкретної програми:
Правою кнопкою на ярлик → **Властивості** → **Сумісність** → **Змінити параметри високого DPI** → **Заміщення масштабування** → **Застосунок**

### Різні частоти оновлення на моніторах

Кожен монітор може мати свою частоту — це нормально. Налаштуй кожен окремо через **Додаткові параметри дисплея**.

### Монітор показує "Немає сигналу"

- Перевір що правильне джерело входу вибрано на самому моніторі (HDMI 1, HDMI 2, DP тощо) — кнопки на корпусі монітора
- Перевір чи кабель підключений до відеокарти, а не до материнської плати (якщо є окрема відеокарта)

## Корисні гарячі клавіші для двох моніторів

| Дія | Клавіші |
|-----|---------|
| Перемістити вікно на інший монітор | `Win + Shift + ←/→` |
| Розгорнути на весь екран | `Win + ↑` |
| Вибрати режим дисплея | `Win + P` |
| Заблокувати ПК | `Win + L` |

---

## Підсумок

`Win + P` → **Розширити** — базове налаштування. Перетягни прямокутники в налаштуваннях відповідно до розташування на столі. Для кожного монітора окремо встанови нативну роздільну здатність і максимальну частоту оновлення.
