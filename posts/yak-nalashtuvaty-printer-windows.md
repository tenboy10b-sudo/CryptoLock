Більшість принтерів встановлюються автоматично. Але якщо щось пішло не так — ось повний покроковий гайд.

---

## Підключення через USB

1. Підключи принтер кабелем до ПК
2. Увімкни принтер
3. Windows автоматично знайде і встановить драйвер

Перевір: `Win + I` → **Bluetooth і пристрої** → **Принтери та сканери** — принтер має з'явитись в списку.

Якщо не з'явився: натисни **Додати пристрій** → Windows знову виконає пошук.

---

## Підключення через Wi-Fi

Принтер і ПК мають бути в одній мережі Wi-Fi.

1. Підключи принтер до Wi-Fi через його меню (зазвичай через кнопку WPS або налаштування на екрані принтера)
2. `Win + I` → **Bluetooth і пристрої** → **Принтери та сканери** → **Додати пристрій**
3. Windows знайде принтер в мережі автоматично

Якщо не знаходить: натисни **Потрібного принтера немає у списку** → **Додати принтер за IP-адресою**.

Дізнатись IP-адресу принтера: надрукуй сторінку конфігурації з меню принтера, або перевір в налаштуваннях роутера.

---

## Встановити принтер за замовчуванням

`Win + I` → **Bluetooth і пристрої** → **Принтери та сканери** → клікни на потрібний принтер → **Встановити за замовчуванням**

Або вимкни автоматичне визначення:
`Win + I` → **Bluetooth і пристрої** → **Принтери та сканери** → вимкни **Дозволити Windows керувати стандартним принтером**

Через PowerShell:
```powershell
# Список принтерів
Get-Printer | Select-Object Name, Default, PrinterStatus

# Встановити за замовчуванням
Set-Printer -Name "HP LaserJet" -Default $true
```

## Якщо драйвер не встановився автоматично

1. Знайди модель принтера на наклейці або в документах
2. Перейди на сайт виробника (hp.com, canon.com, epson.com тощо)
3. Завантаж драйвер для своєї версії Windows
4. Встанови і перезавантаж

Або спробуй через Windows Update:
`Win + I` → **Windows Update** → **Додаткові параметри** → **Необов'язкові оновлення** — тут можуть бути драйвери принтерів.

---

## Типові проблеми

**Принтер показує "Офлайн":**
```powershell
# Перезапустити спулер друку
Restart-Service Spooler
```

Або: `Win + R` → `services.msc` → **Print Spooler** → **Перезапустити**

**Черга друку застрягла:**
```powershell
Stop-Service Spooler
Remove-Item "C:\Windows\System32\spool\PRINTERS\*" -Force
Start-Service Spooler
```

**Принтер друкує але якість погана:** перевір налаштування якості в **Параметрах принтера** і рівень чорнила/тонера.

---

## Спільний доступ до принтера в мережі

Щоб інші ПК в мережі могли друкувати на твоєму принтері:

`Win + I` → **Bluetooth і пристрої** → **Принтери та сканери** → вибери принтер → **Параметри принтера** → вкладка **Спільний доступ** → **Надати спільний доступ до цього принтера**.

---

## Резюме

USB принтер: підключи кабель — Windows встановить автоматично. Wi-Fi принтер: підключи до тієї самої мережі → Додати пристрій. Проблеми з чергою — перезапусти Spooler. Не встановлюється драйвер — завантаж з сайту виробника.
