4 ГБ RAM у 2026 році — це мало. Але перш ніж купувати нову планку, спробуй звільнити пам'ять програмно.

---

## Знайти що займає найбільше RAM

`Ctrl + Shift + Esc` → **Процеси** → клікни на **Пам'ять** щоб відсортувати.

```powershell
# Топ-20 процесів по споживанню RAM
Get-Process | Sort-Object WorkingSet -Descending |
  Select-Object -First 20 Name,
    @{N="RAM MB"; E={[math]::Round($_.WorkingSet/1MB, 0)}} |
  Format-Table -AutoSize
```

Найчастіші пожирачі: Chrome (кожна вкладка ~100–300 МБ), Microsoft Teams, Outlook, VS Code, Discord.

---

## 1. Закрити зайві вкладки в браузері

Один з найшвидших способів звільнити сотні МБ.

**Chrome:** увімкни Memory Saver — вкладки що не активні "засинають":
`chrome://settings` → **Продуктивність** → **Memory Saver** → увімкни.

---

## 2. Вимкнути зайве з автозапуску

Програми в автозапуску завантажуються в пам'ять при старті Windows і залишаються там.

`Ctrl + Shift + Esc` → **Автозапуск** → вимкни все що не потрібне постійно.

---

## 3. Збільшити файл підкачки

Файл підкачки (pagefile.sys) — це частина диска що Windows використовує як "повільний RAM".

`Win + R` → `sysdm.cpl` → **Додатково** → **Параметри** (Швидкодія) → **Додатково** → **Змінити**:

- Зніми **"Автоматично вибирати"**
- Вибери диск C → **Спеціальний розмір**
- Початковий: обсяг RAM × 1.5 (наприклад 6144 МБ для 4 ГБ RAM)
- Максимальний: обсяг RAM × 3 (12288 МБ)
- **Встановити** → **OK** → перезавантаж

---

## 4. Вимкнути фонові служби

```powershell
# Служби що можна вимкнути для економії RAM
$services = @(
    "SysMain",          # Superfetch — кешує дані в RAM (корисний на HDD, менш потрібний на SSD)
    "DiagTrack",        # телеметрія
    "WSearch",          # індексування (якщо не користуєшся пошуком)
    "TabletInputService"# сенсорне введення
)

foreach ($svc in $services) {
    Stop-Service $svc -Force -ErrorAction SilentlyContinue
    Set-Service $svc -StartupType Disabled -ErrorAction SilentlyContinue
}
```

---

## 5. ReadyBoost — флешка як кеш

ReadyBoost дозволяє використовувати USB флешку як додатковий кеш (не RAM, але знижує навантаження на диск).

1. Вставни швидку USB флешку (USB 3.0, мінімум 8 ГБ)
2. Правою кнопкою у Провіднику → **Властивості** → **ReadyBoost**
3. Вибери **"Використовувати цей пристрій"** → встанови максимальний обсяг → **OK**

> Ефективно тільки якщо диск HDD. На SSD ReadyBoost не дає помітного результату.

---

## 6. Вимкнути візуальні ефекти

Анімації і прозорість також споживають пам'ять відеокарти і системну RAM.

`Win + R` → `sysdm.cpl` → **Додатково** → **Параметри** → **Налаштувати для найкращої швидкодії**

---

## 7. Перевірити на витоки пам'яті

Якщо якийсь процес з часом займає все більше RAM — це витік пам'яті.

```powershell
# Відстежити зростання споживання RAM процесу протягом 10 хвилин
$processName = "chrome"
for ($i = 0; $i -lt 10; $i++) {
    $ram = (Get-Process $processName -ErrorAction SilentlyContinue |
      Measure-Object WorkingSet -Sum).Sum / 1MB
    Write-Output "$(Get-Date -Format 'HH:mm:ss') — $processName використовує $([math]::Round($ram, 0)) МБ"
    Start-Sleep -Seconds 60
}
```

## Скільки RAM реально потрібно

| Сценарій | Мінімум | Комфортно |
|----------|---------|-----------|

| Відеоконференції (Zoom/Teams) | 8 ГБ | 16 ГБ |
| Розробка (VS Code + Docker) | 16 ГБ | 32 ГБ |
| Відеомонтаж | 16 ГБ | 32–64 ГБ |

## Підсумок

Найшвидший ефект: закрити зайві вкладки Chrome + вимкнути автозапуск + збільшити файл підкачки. Якщо ПК постійно на межі — єдине реальне рішення це апгрейд RAM. Перевір чи є вільні слоти і яка максимальна підтримувана пам'ять: `Win + R` → `msinfo32` → **Встановлена фізична пам'ять**.
