Windows гальмує, програми закриваються самі — часто причина у нестачі RAM. Ось що можна зробити без покупки нових планок і що робити якщо покупка все ж потрібна.

---

## Перевірити скільки RAM використовується

`Ctrl + Shift + Esc` → **Продуктивність** → **Пам'ять**:
- Показує скільки зайнято і скільки вільно
- **Слоти, що використовуються** — важливо для апгрейду
- **Швидкість** — частота RAM

Якщо вільно менше 500 МБ — RAM не вистачає.

---

## Крок 1: Вивільнити RAM без перезавантаження

```powershell
# Закрити непотрібні процеси що займають багато пам'яті
Get-Process | Sort-Object WorkingSet -Descending |
  Select-Object Name,
    @{N="RAM MB";E={[math]::Round($_.WorkingSet/1MB)}} |
  Select-Object -First 10 |
  Format-Table -AutoSize
```

Знайди процеси що займають найбільше — закрий непотрібні.

---

## Крок 2: Вимкнути зайвий автозапуск

`Ctrl + Shift + Esc` → **Автозапуск** → вимкни програми з позначкою **Високий** вплив.

Кожна фонова програма займає 50–300 МБ RAM.

---

## Крок 3: Збільшити файл підкачки

Файл підкачки — додаткова "повільна RAM" на диску.

`Win + R` → `sysdm.cpl` → **Додатково** → **Параметри** → **Додатково** → **Змінити**:
- Зніми **Автоматично вибирати**
- Вибери диск → **Спеціальний розмір**
- Початковий: RAM × 1.5 (наприклад для 4 ГБ → 6144 МБ)
- Максимальний: RAM × 3 (наприклад 12288 МБ)

> На SSD файл підкачки працює значно швидше ніж на HDD.

---

## Крок 4: Перевірити чи можна додати RAM

**Скільки слотів є і зайнято:**

`Ctrl + Shift + Esc` → Продуктивність → Пам'ять → **Слоти, що використовуються: 2 з 4** — є 2 вільних слоти.

```powershell
# Детальна інформація про RAM
Get-WmiObject Win32_PhysicalMemory |
  Select-Object BankLabel, Capacity, Speed, Manufacturer |
  ForEach-Object {
    [PSCustomObject]@{
      Слот       = $_.BankLabel
      "ГБ"       = [math]::Round($_.Capacity/1GB)
      "МГц"      = $_.Speed
      Виробник   = $_.Manufacturer
    }
  } | Format-Table -AutoSize

# Максимальний підтримуваний обсяг
(Get-WmiObject Win32_PhysicalMemoryArray).MaxCapacity / 1MB
```

## Скільки RAM потрібно

| Використання | Мінімум | Комфортно |
|-------------|---------|-----------|
| Базова робота (браузер, офіс) | 4 ГБ | 8 ГБ |
| Розробка, кілька програм | 8 ГБ | 16 ГБ |
| Відеомонтаж, 3D, ігри | 16 ГБ | 32 ГБ |
| Віртуальні машини, сервер | 16+ ГБ | 32–64 ГБ |

---

## Що купувати

Перевір сумісність перед покупкою:
- Тип: DDR4 або DDR5 (залежить від материнської плати)
- Частота: відповідна підтримуваній материнською платою
- Обсяг: ліпше 2×8 ГБ ніж 1×16 ГБ (dual channel швидший)

Перевір на сайті виробника ноутбука або через CPU-Z (безкоштовна програма).

---

## Підсумок

Спочатку: вимкни автозапуск і закрий фонові програми. Збільш файл підкачки якщо RAM менше 8 ГБ. Якщо є вільні слоти — додай планки: 8→16 ГБ відчувається одразу. Dual channel (2 однакові планки) швидший за single.
