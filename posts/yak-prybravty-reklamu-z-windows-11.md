Windows 11 показує рекламу в меню Пуск, на екрані блокування, у Провіднику і налаштуваннях. Все це вимикається — розповідаємо де.

---

## 1. Реклама в меню Пуск

`Параметри` → `Персоналізація` → `Пуск`:
- Вимкни **"Показувати рекомендації для порад, ярликів, нових і відновлених програм"**
- Вимкни **"Показувати нещодавно додані програми"** (за бажанням)
- Вимкни **"Показувати нещодавно відкриті елементи"** (за бажанням)

---

## 2. Реклама на екрані блокування (Spotlight)

`Параметри` → `Персоналізація` → `Екран блокування`:
- **Персоналізація екрану блокування** → змінити з **Windows Spotlight** на **Зображення** або **Слайд-шоу**

Або через реєстр щоб заблокувати Spotlight назавжди:

```powershell
Set-ItemProperty "HKCU:\SOFTWARE\Policies\Microsoft\Windows\CloudContent" `
  -Name "DisableWindowsSpotlightFeatures" -Value 1 -Type DWord -Force
```

---

## 3. Рекламні поради і підказки

`Параметри` → `Система` → `Сповіщення`:
- Прокрути вниз → вимкни **"Пропонувати поради під час використання Windows"**
- Вимкни **"Отримувати поради і пропозиції під час використання Windows"**

---

## 4. Реклама в Провіднику

Провідник показує рекламу OneDrive і синхронізації.

```powershell
# Вимкнути рекламу в Провіднику
Set-ItemProperty "HKCU:\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced" `
  -Name "ShowSyncProviderNotifications" -Value 0 -Type DWord
```

Або: `Провідник` → `...` → `Параметри` → вкладка **Вигляд** → зніми **"Показувати сповіщення постачальника синхронізації"**

---

## 5. Персоналізована реклама і відстеження

`Параметри` → `Конфіденційність і безпека` → `Загальне`:
- Вимкни **"Дозволити програмам показувати мені персоналізовану рекламу"**
- Вимкни **"Показувати рекомендований вміст у Параметрах"**
- Вимкни **"Дозволити Windows покращувати пуск і результати пошуку"**

---

## 6. Телеметрія і діагностичні дані

`Параметри` → `Конфіденційність і безпека` → `Діагностика і відгуки`:
- **Діагностичні дані** → **Обов'язкові** (мінімум)
- Вимкни **"Покращення рукопису та набору тексту"**
- Вимкни **"Налаштовані можливості"**
- **Видалити діагностичні дані** → натисни **Видалити**

---

## 7. Реклама в Edge і пошуку

`Параметри` → `Конфіденційність і безпека` → `Дозволи для пошуку`:
- Вимкни **"Отримувати результати пошуку з інтернету"** якщо не хочеш онлайн-пошуку в меню Пуск

---

## 8. Масове вимкнення через PowerShell

```powershell
# Вимкнути більшість рекламних функцій одним скриптом
$settings = @{
    # Реклама в меню Пуск
    "HKCU:\SOFTWARE\Microsoft\Windows\CurrentVersion\ContentDeliveryManager" = @{
        "SystemPaneSuggestionsEnabled" = 0
        "SoftLandingEnabled"           = 0
        "SubscribedContentEnabled"     = 0
        "OemPreInstalledAppsEnabled"   = 0
        "PreInstalledAppsEnabled"      = 0
        "SilentInstalledAppsEnabled"   = 0
    }
    # Провідник
    "HKCU:\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced" = @{
        "ShowSyncProviderNotifications" = 0
    }
    # Загальна реклама
    "HKCU:\SOFTWARE\Microsoft\Windows\CurrentVersion\AdvertisingInfo" = @{
        "Enabled" = 0
    }
}

foreach ($path in $settings.Keys) {
    if (-not (Test-Path $path)) { New-Item $path -Force | Out-Null }
    foreach ($name in $settings[$path].Keys) {
        Set-ItemProperty $path -Name $name -Value $settings[$path][$name] -Type DWord -Force
    }
}

Write-Output "✓ Рекламу вимкнено. Перезапусти Провідник або перезавантаж ПК."
```

---

## 9. Перезапустити Провідник

Щоб зміни набули чинності без перезавантаження:

```powershell
Stop-Process -Name explorer -Force
Start-Process explorer
```

---

## Підсумок

Найважливіше: `Параметри` → `Персоналізація` → `Пуск` → вимкни рекомендації. `Конфіденційність` → `Загальне` → вимкни персоналізовану рекламу. Для екрану блокування — змінити з Spotlight на статичне зображення. PowerShell скрипт вище робить все одразу.
