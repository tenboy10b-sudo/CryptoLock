Wi-Fi раптово зник або кнопка вмикання недоступна — типова ситуація після оновлення Windows або встановлення нових драйверів. Розберемо по порядку.

## Крок 1. Перевір фізичний перемикач

На ноутбуках часто є апаратний перемикач Wi-Fi — окрема клавіша або комбінація `Fn + F2/F3/F5` (залежно від моделі). Перевір що він увімкнений.

---

## Крок 2. Увімкни Wi-Fi через Параметри

**Windows 11:**
`Win + I` → **Мережа та Інтернет** → перемикач **Wi-Fi** → увімкни.

**Windows 10:**
`Win + I` → **Мережа та Інтернет** → **Wi-Fi** → увімкни перемикач.

Якщо перемикача Wi-Fi немає взагалі — переходь до кроку 3.

---

## Крок 3. Перевір Диспетчер пристроїв

**Крок 1.** `Win + X` → **Диспетчер пристроїв**.

**Крок 2.** Розкрий розділ **Мережеві адаптери**.

**Крок 3.** Знайди Wi-Fi адаптер (зазвичай містить слова Wireless, Wi-Fi або назву чіпсету: Intel Wireless, Realtek RTL...).

**Якщо є жовтий знак оклику** — проблема з драйвером. Клікни правою кнопкою → **Оновити драйвер** → **Автоматичний пошук**.

**Якщо адаптер є зі стрілкою вниз** — він вимкнений. Клікни правою кнопкою → **Задіяти**.

**Якщо Wi-Fi адаптера немає взагалі** — драйвер не встановлений або адаптер не розпізнається. Дивись крок 5.

---

## Крок 4. Перевір службу автоналаштування WLAN

**Крок 1.** `Win + R` → `services.msc` → Enter.

**Крок 2.** Знайди **Служба автоналаштування WLAN** (WLAN AutoConfig).

**Крок 3.** Тип запуску має бути **Автоматично**, статус — **Виконується**.

**Крок 4.** Якщо зупинена — клікни правою кнопкою → **Запустити**.

---

## Крок 5. Перевстанови драйвер Wi-Fi

**Крок 1.** Дізнайся модель Wi-Fi адаптера: `Win + X` → Диспетчер пристроїв → Мережеві адаптери.

**Крок 2.** Зайди на сайт виробника ноутбука або материнської плати і завантаж актуальний драйвер.

**Крок 3.** В Диспетчері пристроїв клікни правою кнопкою на адаптер → **Видалити пристрій** → постав галочку **Видалити драйвер** → Видалити.

**Крок 4.** Перезавантаж і встанови завантажений драйвер.

---

## Крок 6. Скидання мережевих налаштувань

Відкрий CMD від адміністратора:

```cmd
netsh winsock reset
netsh int ip reset
ipconfig /flushdns
ipconfig /release
ipconfig /renew
```

Перезавантаж після виконання.

---

## Крок 7. Через PowerShell

Перевірити стан Wi-Fi адаптера:

```powershell
Get-NetAdapter | Where-Object {$_.Name -like "*Wi-Fi*" -or $_.Name -like "*Wireless*"}
```

Увімкнути адаптер:
```powershell
Enable-NetAdapter -Name "Wi-Fi"
```
