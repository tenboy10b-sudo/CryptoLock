Помітив що ПК гальмує, сам відкриваються вікна, або браузер перенаправляє на дивні сайти? Це ознаки зараження. Ось покроковий план як позбутись без платних програм.

## Ознаки зараження

- ПК сильно гальмує без видимої причини
- Браузер сам відкриває рекламні сайти
- На робочому столі нові ярлики яких ти не ставив
- Антивірус вимкнувся сам
- В автозавантаженні невідомі програми
- Диск постійно завантажений на 100%

---

## Крок 1. Відключи від інтернету

Одразу вимкни Wi-Fi або виймай кабель. Деякі віруси передають дані або завантажують додаткові компоненти через мережу.

---

## Крок 2. Завантажся в безпечний режим

В безпечному режимі більшість вірусів не запускаються.

`Win + I` → **Система** → **Відновлення** → **Розширений запуск** → **Перезавантажити зараз**.

Або: утримуй `Shift` і натисни **Перезавантаження** на екрані входу.

Після перезавантаження: **Усунення неполадок** → **Додаткові параметри** → **Параметри запуску** → **Перезавантажити** → натисни `F4` для безпечного режиму.

---

## Крок 3. Запусти Windows Defender (повне сканування)

`Win + S` → **Безпека Windows** → **Захист від вірусів і загроз** → **Параметри сканування** → **Повне сканування** → **Сканувати зараз**.

Займає 30–60 хвилин. Усі знайдені загрози — **Видалити**.

### Автономне сканування Defender (найпотужніше)

**Захист від вірусів** → **Параметри сканування** → **Автономне сканування Microsoft Defender** → **Сканувати зараз**.

ПК перезавантажиться і просканує систему ДО завантаження Windows — вірус не зможе сховатись.

---

## Крок 4. Перевір безкоштовним Malwarebytes

Malwarebytes Free знаходить те що пропускає Defender.

1. Зайди на malwarebytes.com → завантаж безкоштовну версію
2. Встанови і запусти **Сканування загроз**
3. Видали все знайдене

---

## Крок 5. Перевір автозавантаження через Autoruns

Завантаж **Autoruns** з learn.microsoft.com/sysinternals → розпакуй → запусти від адміністратора.

Показує **все** що запускається в системі. Підозрілі записи — ті що:
- Мають порожню колонку Description
- Шлях до файлу у `%TEMP%` або `%APPDATA%`
- Не мають Publisher

Клікни правою кнопкою → **Delete** щоб прибрати з автозавантаження.

---

## Крок 6. Очисти браузер

Якщо проблеми в браузері (реклама, перенаправлення):

**Chrome:** Налаштування → Додатково → Скинути налаштування → Відновити вихідні налаштування.

Перевір розширення: `Ctrl + Shift + E` → видали незнайомі.

**Firefox:** Довідка → Інформація для вирішення проблем → Очистити Firefox.

---

## Крок 7. Перевір файл hosts

Деякі віруси змінюють файл hosts щоб перенаправляти сайти.

Відкрий Блокнот від адміністратора → Файл → Відкрити → перейди до:
```
C:\Windows\System32\drivers\etc\hosts
```

Файл повинен містити тільки рядки починаючи з `#`. Якщо є рядки типу `127.0.0.1 google.com` — це вірус. Видали їх і збережи.

---

## Крок 8. Після очищення

1. Зміни паролі від важливих акаунтів (пошта, банк)
2. Перевір не з'явились чи нові акаунти в Windows: `net user`
3. Увімкни захист Windows Defender якщо він був вимкнений
4. Встанови оновлення Windows

---

## Якщо нічого не допомогло

Деякі сучасні віруси (ransomware, rootkit) неможливо видалити з живої системи. В такому випадку єдине рішення — **скидання Windows до заводських** або чиста переустановка.
