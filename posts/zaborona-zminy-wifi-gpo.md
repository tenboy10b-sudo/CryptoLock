В офісному середовищі часто потрібно щоб користувачі підключались тільки до корпоративної мережі і не могли змінювати налаштування Wi-Fi або підключатись до сторонніх точок доступу.

## Спосіб 1: Заборонити зміну мережевих підключень

**Крок 1.** `Win + R` → `gpedit.msc` → Enter.

**Крок 2.** Перейди:
```
Конфігурація користувача
  → Адміністративні шаблони
    → Мережа
      → Мережеві підключення
```

**Крок 3.** Знайди і увімкни наступні політики:

| Політика | Що забороняє |
|---|---|
| **Заборонити доступ до властивостей підключення до локальної мережі** | Не може відкрити властивості мережі |
| **Можливість підключення до Інтернету тільки для адміністраторів** | Тільки адмін може керувати підключеннями |
| **Заборонити видалення та відключення мережевих підключень** | Не може вимкнути адаптер |

---

## Спосіб 2: Заборонити підключення до конкретних або нових Wi-Fi мереж

**Крок 1.** Перейди у `gpedit.msc`:
```
Конфігурація комп'ютера
  → Параметри Windows
    → Параметри безпеки
      → Бездротові мережі (IEEE 802.11) — Політики
```

**Крок 2.** Якщо розділу немає — клікни правою кнопкою → **Створити нову політику бездротової мережі Windows Vista і пізніших версій**.

**Крок 3.** На вкладці **Загальні** → знайди **Підключатися до мереж із списку** і вимкни **Використовувати будь-яку доступну мережу**.

**Крок 4.** На вкладці **Дозволені мережі** → **Додати** → введи SSID корпоративної мережі → збережи.

Тепер Windows підключатиметься тільки до дозволених мереж.

---

## Спосіб 3: Через реєстр

Заборонити зміну підключень через реєстр:

```cmd
reg add "HKCU\SOFTWARE\Policies\Microsoft\Windows\Network Connections" /v NC_ChangeBindState /t REG_DWORD /d 0 /f
reg add "HKCU\SOFTWARE\Policies\Microsoft\Windows\Network Connections" /v NC_DeleteConnection /t REG_DWORD /d 0 /f
```

---

## Спосіб 4: Заблокувати центр мобільності (для ноутбуків)

Користувачі можуть вмикати/вимикати Wi-Fi через Центр мобільності Windows. Щоб заблокувати:

`gpedit.msc` → **Конфігурація користувача** → **Адміністративні шаблони** → **Компоненти Windows** → **Центр мобільності Windows** → **Вимкнути Центр мобільності Windows** → **Включено**.

---

## Перевірка

Зайди під звичайним користувачем і спробуй:
- Відкрити властивості Wi-Fi підключення
- Забути мережу і підключитись до нової
- Вимкнути Wi-Fi адаптер

Всі ці дії мають бути недоступні або вимагати прав адміністратора.
