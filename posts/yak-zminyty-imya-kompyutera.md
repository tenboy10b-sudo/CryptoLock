Ім'я комп'ютера використовується в мережі для ідентифікації пристрою. За замовчуванням Windows задає випадкове ім'я типу `DESKTOP-AB12CD3`. Змінити на зрозуміліше — елементарно.

## Спосіб 1: Через Параметри (найпростіший)

**Windows 11:**
`Win + I` → **Система** → **Про систему** → **Перейменувати цей ПК** → введи нове ім'я → **Далі** → **Перезавантажити зараз**.

**Windows 10:**
`Win + I` → **Система** → **Про систему** → **Перейменувати цей ПК**.

---

## Спосіб 2: Через Панель керування

`Win + R` → `sysdm.cpl` → Enter → вкладка **Ім'я комп'ютера** → **Змінити** → введи нове ім'я → OK → перезавантаж.

---

## Спосіб 3: Через PowerShell (одна команда)

Відкрий PowerShell від адміністратора:

```powershell
Rename-Computer -NewName "OFFICE-PC-01" -Restart
```

Параметр `-Restart` перезавантажить комп'ютер автоматично. Якщо хочеш перезавантажити пізніше — прибери його.

---

## Спосіб 4: Через CMD

```cmd
wmic computersystem where name="%computername%" call rename name="OFFICE-PC-01"
```

Після цього перезавантаж вручну.

---

## Правила для імені комп'ютера

- Максимум 15 символів
- Тільки латинські літери, цифри і дефіс
- Не може починатись або закінчуватись дефісом
- Без пробілів і спеціальних символів

**Приклади зрозумілих імен:**
- `OFFICE-PC-01` — офісний ПК номер 1
- `RECEPTION-PC` — комп'ютер рецепції
- `IVANOV-LAPTOP` — ноутбук конкретного працівника

---

## Перевірити поточне ім'я комп'ютера

```cmd
hostname
```

Або:
```powershell
$env:COMPUTERNAME
```

---

## Важливо

Після перейменування потрібно **перезавантажити** комп'ютер — зміна набуває чинності тільки після перезапуску. Якщо ПК у домені — перейменування може вплинути на доступ до мережевих ресурсів, проконсультуйся з системним адміністратором.
