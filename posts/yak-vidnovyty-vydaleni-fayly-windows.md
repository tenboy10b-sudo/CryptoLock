Випадково видалив важливий файл — не панікуй. Чим швидше діяти, тим більше шансів відновити. Головне правило: **не записуй нічого нового на диск** поки не відновиш.

---

## Крок 1: Перевір Кошик

Найочевидніше але часто забувають.

Двічі клікни на **Кошик** на робочому столі → знайди файл → правою кнопкою → **Відновити**.

Якщо Кошик порожній або файл видалено через `Shift + Delete` — читай далі.

---

## Крок 2: Попередні версії (Тіньові копії)

Windows автоматично створює знімки файлів якщо увімкнена захист системи.

Клікни правою кнопкою на **папку** де був файл → **Властивості** → вкладка **Попередні версії**.

Якщо є версії — вибери потрібну дату → **Відкрити** → скопіюй файл звідти.

**Або через Shadow Explorer** (безкоштовна програма):
1. Завантаж з shadowexplorer.com
2. Вибери диск і дату знімка
3. Знайди файл → правою кнопкою → **Export**

---

## Крок 3: Історія файлів Windows

Якщо була налаштована резервна копія через File History:

`Пуск` → `Параметри` → `Оновлення та безпека` → `Резервне копіювання` → **Інші параметри** → **Відновити файли з поточної резервної копії**

Або знайди папку де був файл у Провіднику → натисни вкладку **Основне** → **Журнал** (History).

---

## Крок 4: Windows File Recovery (вбудована утиліта)

Microsoft випустила безкоштовну консольну утиліту для відновлення видалених файлів з диска.

**Встановлення:**

`Win + S` → шукай **Windows File Recovery** → встанови з Microsoft Store (безкоштовно).

**Базове використання:**

```cmd
rem Відновити конкретний файл з диску C на диск D
winfr C: D: /n \Users\ІМ'Я\Documents\важливий.docx

rem Відновити всі .docx файли з диску C
winfr C: D: /n *.docx

rem Режим extensive — для давно видалених або відформатованих дисків
winfr C: D: /extensive /n \Users\ІМ'Я\Downloads\

rem Відновити папку повністю
winfr C: D: /n \Users\ІМ'Я\Pictures\
```

Відновлені файли збережуться у папку `Recovery_[дата]` на диску D.

## Крок 5: Сторонні програми

Якщо вбудовані засоби не допомогли — спробуй безкоштовні програми:

### Recuva (Piriform)
- Безкоштовна, проста у використанні
- Завантаж з piriform.com/recuva
- Запусти → вибери тип файлів і диск → Scan → вибери файли → Recover

### TestDisk / PhotoRec
- Безкоштовні, відкритий код
- TestDisk — відновлює розділи і завантажувач
- PhotoRec — відновлює фото, відео, документи незалежно від файлової системи
- Завантаж з cgsecurity.org

### R-Studio (платна)
- Найпотужніший інструмент для складних випадків
- Відновлює з RAID, відформатованих дисків, після вірусів

---

## Важливі правила

**Не встановлюй програму відновлення на той самий диск** з якого відновлюєш — нові файли перезапишуть видалені дані. Встановлюй на інший диск або USB.

**Чим менше часу минуло** — тим більше шансів. Після видалення Windows позначає місце як вільне але не стирає дані фізично. Нові файли поступово перезаписують їх.

**SSD vs HDD:** на SSD відновлення складніше. TRIM — механізм що фізично очищає видалені блоки на SSD — може спрацювати через кілька хвилин. На HDD шанси кращі.
