Windows 10 втратила підтримку у жовтні 2025 року. Оновлення до Windows 11 — безкоштовне, якщо комп'ютер відповідає вимогам. Розберемо покроково як це зробити.

---

## Крок 1: Перевір чи підходить твій ПК

### Мінімальні вимоги Windows 11

| Компонент | Вимога |
|-----------|--------|
| Процесор | Intel 8-е покоління+ або AMD Ryzen 2000+ |
| RAM | 4 ГБ |
| Диск | 64 ГБ вільного місця |
| TPM | Версія 2.0 |
| Secure Boot | Потрібен (увімкнений в BIOS) |
| Дисплей | 720p, 9 дюймів+ |

### Офіційна перевірка через PC Health Check

1. Завантаж **PC Health Check** з сайту Microsoft (шукай "PC Health Check app" в Google)
2. Запусти → натисни **"Check now"**
3. Побачиш результат: підходить чи ні і що саме не відповідає вимогам

---

## Крок 2: Увімкни TPM 2.0 якщо він вимкнений

Найчастіша причина чому перевірка провалюється — TPM вимкнений в BIOS, хоча фізично присутній на більшості ПК і ноутбуків після 2017 року.

### Як перевірити статус TPM

`Win + R` → введи `tpm.msc` → Enter

- **"TPM готовий до використання", версія 2.0** — все добре, переходь до кроку 3
- **"Сумісний TPM не знайдено"** — потрібно увімкнути в BIOS
- **Версія 1.2** — деякі ПК підтримують оновлення до 2.0 через BIOS

### Як увімкнути TPM в BIOS

1. Зайди в BIOS: [Як зайти в BIOS](/yak-zayty-v-bios)
2. Знайди налаштування TPM (може називатись по-різному):
   - **Intel:** Security → Intel Platform Trust Technology (PTT) → Enabled
   - **AMD:** Security → AMD fTPM → Enabled
   - або просто: Advanced → TPM Configuration → TPM → Enabled
3. Збережи зміни (F10) і перезавантаж
4. Повтори перевірку через `tpm.msc`

## Крок 4: Зроби резервну копію

Перед будь-яким великим оновленням — обов'язково збережи важливі файли.

Мінімум: скопіюй папки Документи, Фото, Робочий стіл на зовнішній диск або USB-флешку.

## Крок 5: Запусти оновлення

### Спосіб 1: Через Windows Update (рекомендований)

`Пуск` → `Параметри` → `Оновлення та безпека` → `Windows Update` → `Перевірити наявність оновлень`

Якщо ПК підтримується — побачиш пропозицію "Оновлення до Windows 11". Натисни **"Завантажити і встановити"**.

Процес займає 30–60 хвилин. ПК перезавантажиться кілька разів — це нормально.

### Спосіб 2: Через Installation Assistant

Якщо оновлення не з'являється в Windows Update:

1. Пошукай в Google "Windows 11 Installation Assistant"
2. Завантаж з офіційного сайту Microsoft
3. Запусти → прийми умови → натисни **"Install now"**

### Спосіб 3: Через ISO-образ (чиста установка)

Якщо хочеш чисту установку без залишків Windows 10 — дивись: [Як встановити Windows 11 з флешки](/yak-vstanovyty-windows-11-z-fleshky)

## Що збережеться після оновлення

При оновленні через Windows Update або Installation Assistant:
- ✅ Всі файли і папки
- ✅ Встановлені програми
- ✅ Налаштування
- ✅ Активація Windows (ліцензія переноситься автоматично)

При чистій установці з флешки — тільки те що скопіюєш заздалегідь.

---

## Якщо ПК не підтримує Windows 11

Процесор старіший або TPM 2.0 неможливо увімкнути — офіційне оновлення недоступне.

Варіанти розглянуті детальніше: [Windows 10 кінець підтримки — що робити](/windows-10-kinets-pidtrymky-scho-robyty)

## Після оновлення

1. Перевір чи всі програми працюють
2. Оновити драйвери: `Win + X` → `Диспетчер пристроїв` → перевір наявність оновлень
3. Якщо щось пішло не так — протягом 10 днів після оновлення можна повернутись до Windows 10: `Параметри` → `Система` → `Відновлення` → `Повернутись до Windows 10`
