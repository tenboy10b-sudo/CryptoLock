Диск C заповнюється з часом навіть якщо ти нічого не встановлюєш. Тимчасові файли, кеш оновлень, кошик — все це займає гігабайти. У цій статті розберемо як прибрати зайве швидко і безпечно.

## Спосіб 1: Очищення диска (вбудований інструмент)

Це найпростіший спосіб — займає 2 хвилини.

**Крок 1.** Натисни `Win + S` → введи **Очищення диска** → Enter.

**Крок 2.** Вибери диск **C:** → OK.

**Крок 3.** Windows підрахує скільки можна звільнити. Постав галочки навпроти:
- Тимчасові файли Інтернету
- Тимчасові файли
- Кошик
- Мініатюри

**Крок 4.** Натисни **Очистити системні файли** — це дасть ще більше місця (видалить старі версії Windows після оновлень).

**Крок 5.** OK → Видалити файли.

---

## Спосіб 2: Через командний рядок

Відкрий CMD від імені адміністратора (`Win + X` → Термінал (адміністратор)) і виконай:

```cmd
cleanmgr /d C: /verylowdisk
```

Ця команда запускає очищення з максимальними параметрами.

Щоб очистити тимчасові файли вручну:

```cmd
del /q /f /s %TEMP%\*
del /q /f /s C:\Windows\Temp\*
```

Частина файлів може не видалитись бо використовується системою — це нормально, просто пропусти помилки.

---

## Спосіб 3: Через Параметри Windows (найповніший)

**Крок 1.** `Win + I` → **Система** → **Пам'ять**.

**Крок 2.** Windows покаже що займає місце на диску C по категоріях.

**Крок 3.** Натисни **Тимчасові файли** → вибери що видалити → **Видалити файли**.

**Крок 4.** Увімкни **Контроль пам'яті** — Windows буде автоматично чистити тимчасові файли за розкладом.

---

## Що можна видаляти безпечно

| Файли | Безпечно видаляти? |
|---|---|
| Тимчасові файли (`%TEMP%`) | Так |
| Кошик | Так |
| Кеш браузера | Так |
| Старі версії Windows (`Windows.old`) | Так, якщо система стабільна |
| Файли дампу пам'яті | Так |
| Файли журналів | Так |

## Що НЕ видаляти

- Файли в `C:\Windows\System32` — системні файли
- Папка `C:\Users\Ім'я\Documents` — твої документи
- Файли програм в `C:\Program Files`

---

## Скільки місця звільниться

На середньостатистичному ПК після очищення звільняється від 5 до 30 ГБ. Якщо диск все одно повний — перевір що займає місце через **Параметри → Система → Пам'ять** і видали непотрібні програми або перенеси файли на інший диск.
