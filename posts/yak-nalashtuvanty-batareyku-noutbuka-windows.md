Батарея ноутбука деградує з часом. Windows має інструменти щоб відстежити її стан і налаштувати поведінку для максимального терміну служби.

---

## Перевірити здоров'я батареї

Windows вміє генерувати детальний звіт про батарею:

Відкрий **CMD від адміністратора**:

```cmd
powercfg /batteryreport /output C:\battery-report.html
```

Відкрий файл `C:\battery-report.html` у браузері.

Що шукати у звіті:
- **Design Capacity** — проектна ємність (нова батарея)
- **Full Charge Capacity** — поточна ємність при повному заряді
- **Cycle Count** — кількість циклів заряду

**Здоров'я батареї = Full Charge Capacity / Design Capacity × 100%**

Норма: 80%+ — добрий стан. Нижче 60% — варто замінити.

---

## Режими живлення

`Параметри` → `Система` → `Живлення та сон` → **Додаткові параметри живлення**:

- **Збалансований** — автоматичне регулювання, стандарт
- **Економія енергії** — менша продуктивність, більше заряду
- **Висока продуктивність** — максимум, батарея сідає швидше

**Рекомендація:** Збалансований для більшості задач. Економія — якщо далеко від розетки.

---

## Поріг заряду для довшого терміну служби

Літій-іонні батареї живуть довше якщо не заряджати до 100% постійно. Ідеальний діапазон: 20–80%.

### На ноутбуках Lenovo
Встановлюй через **Lenovo Vantage** → Power → Battery Charge Threshold:
- Start charging: 20%
- Stop charging: 80%

### На ноутбуках ASUS
**MyASUS** → Battery Care Mode → вибери 60% або 80% максимум

### На ноутбуках Dell
**Dell Power Manager** → Battery Settings → Primary AC Use

### Універсально через BIOS
Деякі ноутбуки мають налаштування Battery Limit в BIOS.

## Сповіщення про рівень батареї

`Win + R` → `powercfg.cpl` → **Змінити параметри плану** → **Змінити додаткові параметри живлення** → **Батарея**:

- **Рівень низького заряду батареї** → 20%
- **Рівень критичного заряду** → 10%
- **Дія при низькому заряді** → Сповіщення
- **Дія при критичному заряді** → Сон або Гібернація

## Підсвічування і Wi-Fi — найбільші споживачі

```powershell
# Переглянути що споживає найбільше енергії
powercfg /energy /output C:\energy-report.html /duration 60
```

Що реально економить:
- Яскравість екрана: 50% замість 100% → -20–30% споживання
- Wi-Fi вимкнути якщо не потрібен
- Bluetooth вимкнути
- Клавіатурне підсвічування вимкнути

Швидко: `Win + A` → повзунок яскравості.

---

## Режим Battery Saver

Автоматично вмикається при низькому заряді або вручну.

`Параметри` → `Система` → `Живлення та сон` → **Економія заряду батареї**:
- Встанови поріг: 20%
- "Знижувати яскравість екрана" — увімкни

Або швидко: `Win + A` → **Економія заряду**.

---

## Калібрування батареї (для точних показників)

Якщо Windows показує неправильний відсоток — повний цикл заряду/розряду допомагає:

1. Заряди до 100%
2. Від'єднай і розряди до автовимкнення
3. Заряди до 100% не вмикаючи
4. Увімкни

Робити не частіше ніж раз на 3 місяці — часте повне розряджання шкодить літій-іонним батареям.

---

## Підсумок

`powercfg /batteryreport` покаже реальну ємність і здоров'я батареї. Для максимального терміну служби: заряджай в діапазоні 20–80% через програму виробника ноутбука. Яскравість на 50% — найпростіший спосіб суттєво збільшити час роботи.
