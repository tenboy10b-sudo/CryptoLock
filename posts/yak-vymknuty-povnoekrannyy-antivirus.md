Антивірусні сповіщення в невідповідний момент — не привід вимикати захист. Їх можна приглушити правильно.

---

## Вимкнути рекламні сповіщення Defender (не критичні)

`Безпека Windows` → `Налаштування` (шестерня) → `Сповіщення` → вимкни:
- "Надсилати сповіщення безпеки"
- Сповіщення про рекомендації безпеки

**НЕ вимикай:** сповіщення про виявлені загрози і результати сканування.

---

## Focus Assist — блокувати всі сповіщення тимчасово

`Win + A` → **Focus Assist** → вибери **Тільки пріоритетні** або **Тільки будильники**.

Або: `Параметри` → `Система` → `Focus` → налаштуй правила (наприклад вмикати при іграх).

---

## Режим гри автоматично вимикає сповіщення

`Параметри` → `Ігри` → `Режим гри` → увімкни. При запуску гри Windows автоматично пригнічує фонові сповіщення.

---

## Через PowerShell — вимкнути конкретний тип сповіщень

```powershell
# Вимкнути сповіщення Defender про рекомендації
Set-MpPreference -UILockdown $true
# Це вимикає всі UI сповіщення Defender — обережно
```

---

## Підсумок

`Win + A` → **Focus Assist** — найшвидший спосіб на час презентації або гри. В Безпеці Windows → Налаштування → Сповіщення вимикай тільки рекламні нотифікації, не критичні попередження про загрози.
