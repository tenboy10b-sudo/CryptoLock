## Навіщо чистити DNS кеш

DNS кеш — це список сайтів які ти відвідував і їхніх IP-адрес. Іноді він застаріває або пошкоджується, що призводить до:

- Сайти не відкриваються хоча інтернет є
- Старий IP замість нового (сайт переїхав на інший хостинг)
- Незвичні перенаправлення

## Як почистити через CMD

Відкрий **CMD від імені адміністратора** і виконай:

```cmd
ipconfig /flushdns
```

Побачиш повідомлення: `Кеш розпізнавача DNS успішно очищено`.

## Через PowerShell

```powershell
Clear-DnsClientCache
```

Перевір що кеш порожній:

```powershell
Get-DnsClientCache
```

## Бонус: скинути всі мережеві налаштування

Якщо проблеми залишаються — виконай повний скид:

```cmd
ipconfig /release
ipconfig /renew
netsh winsock reset
netsh int ip reset
```

Після цього перезавантаж комп'ютер.
