Іноді потрібно швидко створити обліковий запис адміністратора — для нового співробітника, для резервного доступу або під час відновлення системи. Командний рядок дозволяє зробити це за кілька секунд.

## Крок 1. Відкрий CMD від адміністратора

Натисни `Win + X` → вибери **Термінал Windows (адміністратор)** або **Командний рядок (адміністратор)**.

Або: `Win + S` → введи `cmd` → правою кнопкою → **Запустити від імені адміністратора**.

---

## Крок 2. Створи нового користувача

```cmd
net user ІМ'Я_КОРИСТУВАЧА ПАРОЛЬ /add
```

Наприклад:

```cmd
net user admin2 MyPass2025! /add
```

Правила для пароля:
- Мінімум 8 символів
- Має містити великі і малі літери, цифри
- Пробіли в паролі — беруть в лапки: `"мій пароль"`

Якщо все вірно побачиш: **Команда виконана успішно.**

---

## Крок 3. Додай користувача до групи адміністраторів

```cmd
net localgroup Администраторы ІМ'Я_КОРИСТУВАЧА /add
```

На англомовній Windows:

```cmd
net localgroup Administrators ІМ'Я_КОРИСТУВАЧА /add
```

---

## Крок 4. Перевір що все налаштовано правильно

```cmd
net user ІМ'Я_КОРИСТУВАЧА
```

У рядку **Членство в локальних групах** має бути вказано `*Администраторы` або `*Administrators`.

---

## Додаткові команди

**Переглянути всіх користувачів:**
```cmd
net user
```

**Переглянути всіх адміністраторів:**
```cmd
net localgroup Администраторы
```

**Змінити пароль користувача:**
```cmd
net user ІМ'Я НовийПароль
```

**Вимкнути обліковий запис (не видаляти):**
```cmd
net user ІМ'Я /active:no
```

**Видалити користувача:**
```cmd
net user ІМ'Я /delete
```

---

## Створення через PowerShell

Якщо віддаєш перевагу PowerShell:

```powershell
$Password = ConvertTo-SecureString "MyPass2025!" -AsPlainText -Force
New-LocalUser -Name "admin2" -Password $Password -FullName "Адміністратор 2"
Add-LocalGroupMember -Group "Administrators" -Member "admin2"
```

---

## Активувати вбудований обліковий запис адміністратора

В Windows є прихований обліковий запис `Administrator` який за замовчуванням вимкнений. Щоб увімкнути:

```cmd
net user Administrator /active:yes
net user Administrator НовийПароль
```

> **Обережно:** вбудований адміністратор не має UAC-підтвердження — всі програми запускаються з повними правами автоматично. Використовуй тільки коли потрібно.
