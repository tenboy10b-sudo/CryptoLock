Резервна копія окремих файлів — добре. Але якщо Windows перестала завантажуватись або диск вийшов з ладу — потрібен повний образ системи. Ось як створити і відновити.

---

## Різниця між образом системи і резервною копією файлів

| | Образ системи | Резервна копія файлів |
|---|---|---|
| Що включає | Весь диск з Windows і програмами | Тільки вибрані файли |
| Розмір | 20–100 ГБ | Залежить від файлів |
| Відновлення | Повне відновлення ОС | Тільки файли |
| Час створення | 15–60 хвилин | Залежить від файлів |

---

## Спосіб 1: Вбудований Windows Backup (Image)

`Панель керування` → `Резервне копіювання і відновлення (Windows 7)` → **Створити образ системи**

Вибери:
- На жорсткому диску (зовнішньому або другому внутрішньому)
- На мережевому розташуванні
- На DVD дисках

Натисни **Почати резервне копіювання**.

---

## Спосіб 2: DISM — через командний рядок

```powershell
# Створити WIM образ системного розділу
# Потрібен запуск від адміністратора і достатньо місця на D:
DISM /Capture-Image `
  /ImageFile:"D:\WindowsBackup.wim" `
  /CaptureDir:"C:\" `
  /Name:"Windows Backup $(Get-Date -Format 'yyyy-MM-dd')" `
  /Compress:max `
  /CheckIntegrity

# Переглянути вміст образу
DISM /Get-ImageInfo /ImageFile:"D:\WindowsBackup.wim"
```

---

## Спосіб 3: Macrium Reflect Free (рекомендовано)

Найпростіший і надійний безкоштовний інструмент.

1. Завантаж з macrium.com → безкоштовна версія
2. Встанови і запусти
3. **Create a backup** → **Image this disk**
4. Вибери диск і папку призначення
5. Натисни **Next** → **Finish**

**Переваги перед вбудованим:**
- Підтримує інкрементальні копії (тільки зміни)
- Можна створити завантажувальний USB для відновлення
- Стискає образ ефективніше

---

## Створити завантажувальний USB для відновлення

**Через Macrium Reflect:**
Other Tasks → **Create Rescue Media** → вибери USB флешку → Finish.

**Через Windows:**
`Панель керування` → `Резервне копіювання` → **Створити диск відновлення системи** → вибери USB.

---

## Відновлення з образу системи

### Через середовище відновлення Windows

При завантаженні → `F8` або 3 рази перервати завантаження кнопкою живлення → **Усунення несправностей** → **Додаткові параметри** → **Відновлення образу системи**

Або завантажся з USB диска відновлення.

### Через DISM (для окремих файлів з образу)

```powershell
# Підключити образ для перегляду (без запису)
DISM /Mount-Image `
  /ImageFile:"D:\WindowsBackup.wim" `
  /Index:1 `
  /MountDir:"C:\MountedImage" `
  /ReadOnly

# Скопіювати потрібний файл
Copy-Item "C:\MountedImage\Windows\System32\ntdll.dll" "C:\Windows\System32\"

# Відключити образ
DISM /Unmount-Image /MountDir:"C:\MountedImage" /Discard
```

## Розклад автоматичного резервного копіювання

```powershell
# Через Task Scheduler — створити образ щотижня у неділю о 2:00
$action = New-ScheduledTaskAction `
  -Execute "wbAdmin.exe" `
  -Argument "start backup -backupTarget:D: -include:C: -allCritical -quiet"

$trigger = New-ScheduledTaskTrigger -Weekly -DaysOfWeek Sunday -At "02:00"

Register-ScheduledTask `
  -TaskName "Weekly System Image" `
  -Action $action `
  -Trigger $trigger `
  -RunLevel Highest `
  -Force
```

## Скільки місця потрібно

Образ стиснутого системного розділу зазвичай:
- Свіжа Windows 11: ~15–20 ГБ
- З програмами і даними: 30–80 ГБ

Рекомендую зовнішній диск мінімум вдвічі більший за системний розділ.

## Підсумок

Для домашнього використання — Macrium Reflect Free щотижня на зовнішній диск. Обов'язково створи **завантажувальний USB** — без нього образ марний якщо Windows не запускається. Для швидкого відновлення окремих файлів — `DISM /Mount-Image` дозволяє дістати конкретний файл не розгортаючи весь образ.
