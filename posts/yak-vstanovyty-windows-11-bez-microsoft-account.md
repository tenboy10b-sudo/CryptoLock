Windows 11 наполегливо вимагає Microsoft акаунт при встановленні. Але встановити з локальним акаунтом можна — є кілька надійних способів.

---

## Спосіб 1: OOBE\BYPASSNRO (найпростіший)

Коли Windows просить підключитись до інтернету або увійти в акаунт:

1. Натисни `Shift + F10` — відкриється командний рядок
2. Введи:

```cmd
oobe\bypassnro
```

3. ПК автоматично перезавантажиться
4. На наступному екрані з'явиться кнопка **"У мене немає інтернету"**
5. Натисни → **"Продовжити з обмеженим налаштуванням"**
6. Введи ім'я користувача і пароль — локальний акаунт створено

---

## Спосіб 2: Вимкнути мережу через термінал

Якщо `bypassnro` не спрацював (буває на нових збірках):

1. `Shift + F10` → CMD
2. Введи:

```cmd
taskkill /F /IM oobenetworkconnectionflow.exe
```

Windows перейде до наступного кроку без перевірки мережі.

---

## Спосіб 3: Фейкові дані (для Home)

На екрані входу в Microsoft акаунт:

1. Введи email: `no@thankyou.com`
2. Будь-який пароль
3. Windows скаже "Щось пішло не так" — натисни **"Далі"**
4. Тепер запропонує створити локальний акаунт

---

## Спосіб 4: Вимкнути мережевий адаптер в BIOS або фізично

Перед початком встановлення від'єднай кабель або вимкни Wi-Fi в BIOS. Windows 11 Home без інтернету автоматично переходить до локального акаунту.

> На Windows 11 Pro натисни **"Приєднатись до домену"** замість входу в Microsoft акаунт — це теж дає локальний акаунт.

---

## Підсумок

`oobe\bypassnro` через `Shift + F10` — найнадійніший спосіб. Для Pro — кнопка "Приєднатись до домену" на екрані входу дає локальний акаунт без хаків.
