Комп'ютер без блокування екрана — це відчинені двері. Якщо відійшов на 5 хвилин, будь-хто може отримати доступ до файлів і облікових записів. Налаштовується за 2 хвилини.

---

## Швидке блокування вручну

Перш ніж налаштовувати автоматику — запам'ятай як заблокувати миттєво:

- **`Win + L`** — найшвидший спосіб, блокує одразу
- `Ctrl + Alt + Del` → **Заблокувати**
- Клік на аватар у меню Пуск → **Заблокувати**

---

## Спосіб 1: Через Параметри (найпростіший)

`Пуск` → `Параметри` → `Система` → `Живлення та сон`

- **Екран** → встанови час після якого екран вимикається (1–5 хвилин для офісу, 10–15 для вдома)

Але це тільки вимикає екран — не блокує обліковий запис. Щоб при увімкненні екрана запитував пароль:

`Параметри` → `Облікові записи` → `Параметри входу` → знайди **Вимагати входу** → вибери **Коли ПК виходить зі сну**

---

## Спосіб 2: Через заставку з паролем

`Win + R` → `control desk.cpl,,@screensaver` → Enter

Або: правою кнопкою на робочому столі → **Персоналізація** → **Заставка**

Налаштуй:
- Заставку (можна вибрати "Порожня" — просто чорний екран)
- **Зачекати:** 5–10 хвилин
- Постав галочку ✅ **При поверненні відображати екран входу**

---

## Спосіб 3: Через реєстр (точне налаштування)

Відкрий `regedit` від адміністратора:

```
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System
```

Знайди або створи параметр `InactivityTimeoutSecs` (тип DWORD):
- Значення в секундах: 300 = 5 хвилин, 600 = 10 хвилин, 900 = 15 хвилин
- Поставити у десятковому форматі

Після встановлення Windows автоматично блокується через заданий час бездіяльності.

---

## Спосіб 4: Через групову політику (для адміністраторів)

`Win + R` → `gpedit.msc`

**Конфігурація комп'ютера** → **Параметри Windows** → **Параметри безпеки** → **Локальні політики** → **Параметри безпеки**:

- **Інтерактивний вхід: обмеження бездіяльності комп'ютера** → встанови в секундах

Для домену через GPMC: [Автоматичне блокування через GPO](/avto-blokuvannya-ekranu-gpo)

---

## Спосіб 5: Через динамічне блокування (Windows Hello)

Якщо є смартфон з Bluetooth — Windows може блокуватись автоматично коли ти йдеш від комп'ютера:

`Параметри` → `Облікові записи` → `Параметри входу` → **Динамічне блокування**:
1. Встанови галочку **Дозволити Windows автоматично блокувати пристрій у вашу відсутність**
2. Спаруй смартфон через Bluetooth

Коли ти відходиш і смартфон виходить за межі Bluetooth-зони — ПК блокується автоматично.

---

## Рекомендації

| Місце використання | Час блокування |
|-------------------|---------------|
| Офіс, відкритий простір | 1–3 хвилини |
| Домашній кабінет | 5–10 хвилин |
| Домашній ПК загального доступу | 3–5 хвилин |
| Сервер або робоча станція | 5 хвилин |

---

## Якщо треба заблокувати всіх користувачів корпоративно

Встанови через групову політику мінімальний час бездіяльності для всього домену. Також налаштуй мінімальну довжину пароля: [Мінімальна довжина пароля в Windows](/minimalna-dovzhyna-parolyu)
