Windows Terminal — сучасний термінал від Microsoft що об'єднує CMD, PowerShell і WSL в одному вікні з вкладками. Якщо ще не використовуєш — варто перейти.

---

## Встановлення

Windows 11 — вже встановлено за замовчуванням.

Windows 10:
```powershell
winget install Microsoft.WindowsTerminal
```

Або з Microsoft Store — безкоштовно.

---

## Основні гарячі клавіші

| Дія | Клавіші |
|-----|---------|
| Нова вкладка | `Ctrl + Shift + T` |
| Закрити вкладку | `Ctrl + Shift + W` |
| Перемикання між вкладками | `Ctrl + Tab` |
| Розділити панель вертикально | `Alt + Shift + +` |
| Розділити панель горизонтально | `Alt + Shift + -` |
| Збільшити шрифт | `Ctrl + =` |
| Новий профіль (вибір) | `Ctrl + Shift + Space` |

## Налаштування профілів

`Ctrl + ,` → відкривається JSON конфіг або GUI налаштувань.

Корисні налаштування для кожного профілю:

```json
{
  "profiles": {
    "defaults": {
      "font": {
        "face": "Cascadia Code",
        "size": 12
      },
      "opacity": 95,
      "useAcrylic": true,
      "scrollbarState": "hidden"
    }
  }
}
```

**Cascadia Code** — шрифт від Microsoft з лігатурами для коду. Встановлюється разом з Terminal.

## Запуск від адміністратора

Правий клік на Terminal в меню Пуск → **Запустити від імені адміністратора**.

Або налаштуй профіль щоб завжди запускався з правами адміна:

Налаштування → вибери профіль → **Запускати від імені адміністратора** → увімкни.

---

## Кольорові схеми

Terminal має вбудовані схеми: One Half Dark, Solarized, Tango. Змінити:

Налаштування → **Кольорові схеми** → вибери або створи свою.

Популярні сторонні схеми: [windowsterminalthemes.dev](https://windowsterminalthemes.dev) — скопіюй JSON і встав в конфіг.

---

## Встановити як термінал за замовчуванням

Windows 11: `Win + I` → **Конфіденційність і безпека** → **Для розробників** → **Термінал** → вибери **Windows Terminal**.

Тепер при запуску PowerShell або CMD вони відкриваються в Terminal.

---

## Резюме

Windows Terminal + PowerShell — стандартна зв'язка для роботи з Windows. Вкладки, розділені панелі і нормальна підтримка Unicode роблять його значно зручнішим за звичайний CMD. Встанови Cascadia Code і налаштуй прозорість — і термінал стане приємним місцем для роботи.
