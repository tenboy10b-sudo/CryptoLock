Диск з драйвером загублений або ноутбук без CD-приводу — це не проблема. Windows вміє знаходити і встановлювати драйвери принтера автоматично.

---

## Спосіб 1: Підключи через USB і зачекай

Найпростіший варіант для USB-принтерів.

1. Підключи принтер USB-кабелем і увімкни його
2. Зачекай 2–3 хвилини — Windows автоматично шукає драйвер через Windows Update
3. Перевір: `Параметри` → `Bluetooth і пристрої` → `Принтери та сканери` — принтер має з'явитись

Якщо з'явився — надрукуй тестову сторінку щоб перевірити.

---

## Спосіб 2: Додати принтер вручну

`Параметри` → `Bluetooth і пристрої` → `Принтери та сканери` → **Додати пристрій**

Windows почне пошук. Якщо принтер знайшовся — клікни **Додати пристрій**.

Якщо не знайшовся — натисни **Потрібного принтера немає в списку** → далі за майстром.

---

## Спосіб 3: Завантажити драйвер з сайту виробника

Найнадійніший спосіб — завжди актуальна версія.

Знайди модель принтера (наклейка знизу або збоку) і зайди на сайт:

| Виробник | Сайт |
|----------|------|
| HP | support.hp.com |
| Canon | usa.canon.com/support |
| Epson | epson.com/support |
| Brother | support.brother.com |
| Samsung | samsung.com/support |
| Kyocera | kyoceradocumentsolutions.eu |

Введи модель → Drivers & Downloads → вибери Windows 10 або 11 64-bit → завантаж і встанови.

---

## Спосіб 4: Універсальні драйвери

Якщо рідкісна модель і рідний драйвер не знаходиться:

**HP Universal Print Driver (UPD)** — працює з більшістю принтерів HP:
`support.hp.com` → пошук "HP Universal Print Driver"

**PCL6 або PostScript Universal Driver** — для офісних принтерів багатьох виробників.

---

## Спосіб 5: Через PowerShell і PnPUtil

```powershell
# Пошук підключених принтерів
Get-PnpDevice | Where-Object { $_.Class -eq "Printer" }

# Оновити драйвер через Windows Update
pnputil /scan-devices

# Встановити .inf драйвер вручну
pnputil /add-driver "C:\Drivers\printer.inf" /install
```

## Типові проблеми

### Принтер встановлений але не друкує

Перевір чергу друку: `Параметри` → принтер → **Відкрити чергу** → видали завислі завдання.

```cmd
net stop spooler
del /Q /F /S "%systemroot%\System32\spool\PRINTERS\*.*"
net start spooler
```

### Принтер підключений але Windows його не бачить

`Win + X` → `Диспетчер пристроїв` → **Принтери** або **Інші пристрої** — чи є жовтий трикутник?

Якщо є — клікни правою → **Оновити драйвер**.

### Помилка "Драйвер недоступний"

Старий драйвер не сумісний з Windows 11. Завантаж нову версію з сайту виробника або спробуй режим сумісності:

Правою кнопкою на інсталятор → **Властивості** → **Сумісність** → запусти для Windows 10.

## Підсумок

Спочатку просто підключи USB і зачекай — Windows сама знайде більшість драйверів. Якщо ні — сайт виробника за моделлю принтера. Черга зависла — `net stop spooler`, очисти папку PRINTERS, `net start spooler`.
