Windows Hello дозволяє входити в систему за відбитком пальця, обличчям або коротким PIN замість довгого пароля. Це швидше і безпечніше — біометричні дані зберігаються локально на пристрої і не передаються в мережу.

---

## Налаштування PIN

PIN — найпростіший варіант, не потребує додаткового обладнання.

`Win + I` → **Облікові записи** → **Параметри входу** → **PIN (Windows Hello)** → **Налаштувати**

Введи поточний пароль облікового запису → задай PIN (мінімум 4 цифри, можна додати літери і символи).

PIN прив'язаний до конкретного пристрою — навіть якщо хтось дізнається твій PIN, він не зможе увійти на іншому ПК.

---

## Налаштування відбитка пальця

Потребує сканера відбитків (вбудованого або USB).

`Win + I` → **Облікові записи** → **Параметри входу** → **Розпізнавання відбитків пальців (Windows Hello)** → **Налаштувати**

Слідуй інструкції — прикладай палець кілька разів під різними кутами. Додай 2–3 пальці для надійності (в тому числі з іншої руки).

---

## Налаштування розпізнавання обличчя

Потребує IR-камери (звичайна веб-камера не підходить). Є на більшості сучасних ноутбуків.

`Win + I` → **Облікові записи** → **Параметри входу** → **Розпізнавання обличчя (Windows Hello)** → **Налаштувати**

Дивись прямо в камеру. Процес займає 10–15 секунд.

Для кращого розпізнавання в окулярах: після першого налаштування натисни **Покращити розпізнавання** і пройди процес ще раз в окулярах.

---

## Вимагати Windows Hello для входу

Щоб відключити можливість входу паролем:

`Win + I` → **Облікові записи** → **Параметри входу** → увімкни **Для підвищення безпеки дозволяти вхід через Windows Hello лише для облікових записів Microsoft на цьому пристрої**

## Управління через PowerShell

```powershell
# Перевірити чи увімкнено Windows Hello
Get-WinUserLanguageList

# Перевірити доступні методи входу
(Get-LocalUser -Name $env:USERNAME).PasswordRequired
```

## Якщо Windows Hello не працює

**Відбиток не розпізнається:**
- Протри сканер сухою тканиною
- Перевстанови відбиток: видали поточний → додай знову
- Оновити драйвер сканера через Device Manager

**Обличчя не розпізнається:**
- Перевір чи не перекрита камера
- Переналаштуй при різному освітленні
- Оновити драйвер камери

**Опції Windows Hello сірі:**
```powershell
# Перевірити чи не заблоковано GPO
Get-ItemProperty "HKLM:\SOFTWARE\Policies\Microsoft\PassportForWork" -ErrorAction SilentlyContinue
```

Якщо є ключ `Enabled = 0` — Windows Hello заблоковано груповою політикою (актуально для корпоративних ПК).

---

## Резюме

Для дому: налаштуй PIN + відбиток або обличчя залежно від обладнання. PIN обов'язковий як запасний метод — він потрібен при перезавантаженні і якщо біометрія не спрацювала. Біометричні дані зберігаються в захищеному сховищі TPM і не виходять за межі пристрою.
