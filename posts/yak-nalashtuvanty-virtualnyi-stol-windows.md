Віртуальні столи дозволяють розділити роботу по темах: один стіл для браузера, другий для коду, третій для месенджерів.

---

## Гарячі клавіші

| Дія | Клавіші |
|-----|---------|
| Огляд всіх столів | `Win + Tab` |
| Новий стіл | `Win + Ctrl + D` |
| Наступний стіл | `Win + Ctrl + →` |
| Попередній стіл | `Win + Ctrl + ←` |
| Закрити стіл | `Win + Ctrl + F4` |
| Перемістити вікно на інший стіл | `Win + Tab` → перетягни |

---

## Через Win + Tab

`Win + Tab` відкриває огляд. Вгорі — всі столи. Внизу — всі вікна поточного столу.

Правою кнопкою на вікно → **Перемістити на** → вибери стіл.

---

## Перейменувати стіл

`Win + Tab` → клікни на назву столу зверху → введи нову назву (наприклад "Робота", "Особисте").

---

## Підсумок

`Win + Ctrl + D` — новий стіл. `Win + Ctrl + ←/→` — перемикання. Корисно коли одночасно відкрито 10+ вікон різних задач.
