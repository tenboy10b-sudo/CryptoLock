Ігровий ноутбук з коробки часто обмежений для економії батареї. Ось як розблокувати повну продуктивність.

---

## 1. План живлення — Висока продуктивність

```powershell
powercfg /setactive 8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c
```

Або через фірмову утиліту: ASUS Armoury Crate, Lenovo Vantage, MSI Center, Dell G-Series → вибери Performance/Turbo режим.

---

## 2. Режим гри і HAGS

`Параметри` → `Ігри` → **Режим гри** → увімкни.

`Параметри` → `Система` → `Дисплей` → `Графіка` → **Hardware-accelerated GPU Scheduling** → увімкни → перезавантаж.

---

## 3. NVIDIA/AMD налаштування

**NVIDIA Control Panel** → Manage 3D Settings → Power management mode → **Prefer maximum performance**.

**AMD Radeon Software** → Gaming → Global Graphics → Power Tuning → Performance.

---

## 4. Вибір дискретної GPU для ігор

`Параметри` → `Система` → `Дисплей` → `Графіка` → вибери програму → **Параметри** → **Висока продуктивність** (дискретна GPU).

## 6. Вимкнути Game Bar і DVR

`Параметри` → `Ігри` → **Xbox Game Bar** → вимкни.

`Параметри` → `Ігри` → **Записи** → вимкни.

---

## Підсумок

Головне: план живлення Висока продуктивність + Performance режим у фірмовій утиліті + NVIDIA/AMD на максимальну продуктивність + Game Mode. Разом дають 15-25% приросту FPS без заміни заліза.
