Принтер не друкує або Windows його не бачить — одна з найпоширеніших проблем. Розберемо підключення з нуля і типові проблеми.

---

## Спосіб 1: Підключення через USB

Найпростіший варіант.

1. Підключи USB-кабель від принтера до комп'ютера
2. Увімкни принтер
3. Windows зазвичай автоматично встановлює драйвер через Windows Update — зачекай 1–2 хвилини

**Перевір чи принтер з'явився:**

`Пуск` → `Параметри` → `Пристрої` → `Принтери та сканери`

Якщо з'явився — натисни на нього → **Керувати** → **Друк тестової сторінки**.

**Якщо не з'явився автоматично — встанови драйвер вручну:**

1. Знайди модель принтера (написано на корпусі або знизу)
2. Зайди на сайт виробника: hp.com, canon.com, epson.com, brother.com
3. Знайди розділ **Підтримка → Драйвери** → введи модель → завантаж і встанови

## Спосіб 2: Підключення через Wi-Fi

Більшість сучасних принтерів підтримують бездротове підключення.

### Через WPS (найшвидше)

1. На принтері знайди кнопку **WPS** (зазвичай зі значком хвиль)
2. На роутері натисни кнопку **WPS**
3. Утримуй кнопку WPS на принтері 2–3 секунди
4. Принтер підключиться до Wi-Fi автоматично

### Через меню принтера

1. На принтері знайди меню **Мережа** або **Wi-Fi Setup**
2. Вибери свою Wi-Fi мережу → введи пароль
3. Після підключення принтер отримає IP-адресу

### Додати принтер у Windows після підключення до Wi-Fi

`Пуск` → `Параметри` → `Пристрої` → `Принтери та сканери` → **Додати принтер або сканер**

Windows знайде принтер у мережі автоматично. Якщо не знайшов — натисни **"Потрібного принтера немає в списку"** → **"Додати принтер за IP-адресою"** → введи IP принтера.

**Як дізнатись IP принтера:** надрукуй сторінку конфігурації — зазвичай кнопка на принтері утримується 5–10 секунд.

## Спосіб 3: Підключення мережевого принтера (в офісі)

Якщо принтер підключений до іншого комп'ютера або сервера в мережі:

`Пуск` → `Параметри` → `Пристрої` → `Принтери та сканери` → **Додати принтер** → **"Потрібного принтера немає в списку"** → **"Вибрати спільний принтер за іменем"** → введи:

```
\\ім'я-комп'ютера\ім'я-принтера
```

або

```
\\192.168.1.100\HP_LaserJet
```

## Типові проблеми

### Принтер є в списку але не друкує

**Перевір чергу друку:**

`Пуск` → `Параметри` → `Пристрої` → `Принтери та сканери` → вибери принтер → **Відкрити чергу**

Якщо в черзі є зависле завдання — клікни правою кнопкою → **Скасувати**.

**Якщо черга не очищається — перезапусти службу:**

Відкрий CMD від адміністратора:

```cmd
net stop spooler
del /Q /F /S "%systemroot%\System32\spool\PRINTERS\*.*"
net start spooler
```

### Принтер показує "Офлайн"

1. Перевір що принтер увімкнений і підключений
2. `Параметри` → `Принтери та сканери` → вибери принтер → **Керувати** → зніми галочку **"Використовувати принтер в автономному режимі"**

### Поганий друк або смуги

- Запусти очищення головки через меню принтера (зазвичай Обслуговування → Очищення головки)
- Перевір рівень чорнила або тонера

### Принтер не бачить Windows після оновлення

Оновлення Windows іноді ламає драйвери. Вирішення: видали принтер зі списку і додай заново, або завантаж новий драйвер з сайту виробника.

## Підсумок

USB-підключення — найнадійніше, Wi-Fi — найзручніше. Більшість проблем з друком вирішуються очищенням черги або перевстановленням драйвера.
