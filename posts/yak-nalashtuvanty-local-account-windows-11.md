Windows 11 при установці наполегливо пропонує увійти через обліковий запис Microsoft. Але локальний акаунт — без прив'язки до хмари — досі можливий. Ось три способи.

---

## Чому людям потрібен локальний акаунт

- Не хочуть прив'язувати ПК до Microsoft акаунту
- Немає інтернету при установці
- Корпоративна політика вимагає локальні акаунти
- Дитячий або спільний ПК без хмарної синхронізації

---

## Спосіб 1: При установці Windows 11 — обхід без інтернету

Під час початкового налаштування коли Windows просить підключитись до Wi-Fi або увійти в акаунт Microsoft:

1. Натисни `Shift + F10` — відкриється CMD
2. Введи:

```cmd
oobe\bypassnro
```

3. ПК перезавантажиться і поверне на початок налаштування
4. Тепер з'явиться кнопка **"У мене немає інтернету"** → **"Продовжити з обмеженим налаштуванням"**
5. Введи ім'я користувача і пароль — локальний акаунт створено

---

## Спосіб 2: Через CMD після входу в систему

Якщо вже увійшов через Microsoft акаунт і хочеш додати локального користувача:

Відкрий CMD від адміністратора:

```cmd
net user ІМ'Я_КОРИСТУВАЧА ПАРОЛЬ /add
net localgroup administrators ІМ'Я_КОРИСТУВАЧА /add
```

Приклад:

```cmd
net user Vasyl Password123 /add
net localgroup administrators Vasyl /add
```

Детальніше: [Як створити адміністратора через CMD](/stvorenny-admina-cherez-cmd)

---

## Спосіб 3: Через Параметри Windows

`Пуск` → `Параметри` → `Облікові записи` → `Сім'я і інші користувачі` → **Додати іншого користувача**

→ У вікні входу натисни **"У мене немає даних для входу цієї особи"**

→ **"Додати користувача без облікового запису Microsoft"**

→ Введи ім'я і пароль → **Далі**

Після створення можна підвищити до адміністратора:

`Параметри` → `Облікові записи` → `Сім'я і інші користувачі` → клікни на новий акаунт → **Змінити тип облікового запису** → **Адміністратор**

---

## Як перейти з Microsoft акаунту на локальний

Якщо вже використовуєш Microsoft акаунт і хочеш відв'язатись:

`Пуск` → `Параметри` → `Облікові записи` → `Ваші дані` → **Увійти натомість за допомогою локального облікового запису**

→ Введи пароль від Microsoft акаунту для підтвердження

→ Введи ім'я і пароль для нового локального акаунту

→ **Вийти і завершити**

> Всі файли і налаштування залишаться на місці. Тільки синхронізація з хмарою Microsoft припиниться.

---

## Різниця між локальним і Microsoft акаунтом

| | Локальний | Microsoft акаунт |
|---|---|---|
| Синхронізація налаштувань | Немає | Так |
| Microsoft Store | Обмежено | Повний доступ |
| OneDrive | Вручну | Автоматично |
| Відновлення пароля | Тільки питання безпеки | Через email |
| Приватність | Вища | Дані в хмарі |

---

## Пароль для локального акаунту

Обов'язково встанови надійний пароль. Без пароля будь-хто хто має фізичний доступ до ПК — автоматично входить в систему.
