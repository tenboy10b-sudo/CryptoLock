Локальний акаунт зберігає дані тільки на ПК — без прив'язки до Microsoft. Корисно для спільних комп'ютерів, кіоскових режимів або коли не хочеш синхронізацію з хмарою.

---

## Створити локальний акаунт через Параметри

`Параметри` → `Облікові записи` → `Сім'я та інші користувачі` → **Додати обліковий запис**:

У вікні входу → **У мене немає даних для входу цієї особи** → **Додати користувача без облікового запису Microsoft** → введи ім'я, пароль і секретні питання.

---

## Створити через CMD

```cmd
rem Створити нового користувача
net user UserName Password123 /add

rem Додати в групу Users (звичайний користувач)
net localgroup Users UserName /add

rem Зробити адміністратором
net localgroup Administrators UserName /add

rem Список всіх облікових записів
net user
```

---

## Створити через PowerShell

```powershell
# Створити локального користувача
New-LocalUser -Name "WorkUser" `
  -Password (ConvertTo-SecureString "Pass123!" -AsPlainText -Force) `
  -FullName "Робочий акаунт" `
  -Description "Акаунт для роботи" `
  -PasswordNeverExpires $false `
  -UserMayNotChangePassword $false

# Додати в групу Users
Add-LocalGroupMember -Group "Users" -Member "WorkUser"

# Переглянути всіх локальних користувачів
Get-LocalUser | Select-Object Name, Enabled, LastLogon
```

---

## Переключитись з Microsoft акаунту на локальний

`Параметри` → `Облікові записи` → `Ваші дані` → **Натомість увійдіть за допомогою локального облікового запису** → введи пароль Microsoft → задай ім'я і пароль локального акаунту → **Вийти та завершити**.

---

## Управління через netplwiz

`Win + R` → `netplwiz` — простий графічний інтерфейс для:
- Додавання і видалення акаунтів
- Зміни типу (стандартний / адміністратор)
- Вимкнення вимоги вводити пароль при вході

---

## Видалити обліковий запис

```powershell
# Видалити користувача (дані залишаться)
Remove-LocalUser -Name "WorkUser"

# Або через net user
net user WorkUser /delete

# Через Параметри: Облікові записи → Сім'я та інші → вибери → Видалити
```

```
"Для використання цього комп'ютера потрібно ввести ім'я і пароль"
→ введи пароль для підтвердження
```

## Підсумок

`net user UserName Password /add` + `net localgroup Users UserName /add` — найшвидший спосіб. `netplwiz` — зручний GUI. Для переходу з Microsoft акаунту на локальний — `Параметри` → `Облікові записи` → `Ваші дані`.
